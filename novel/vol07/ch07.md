### daemon.sh

```bash
# genesis 컨테이너 내부의 daemon.sh 확인
docker exec genesis_core cat /app/daemon.sh
```

```bash
#!/bin/bash
# daemon.sh — genesis Agent Daemon
# Created: 2006-03-15
# Last modified: 2026-01-15
# Author: parkbakha
#
# 이 스크립트는 genesis 컨테이너의 ENTRYPOINT.
# 컨테이너가 시작되면 자동 실행, 종료될 때까지 영구 동작.

set -euo pipefail

LOG_DIR="/data/logs"
DB_PATH="/data/genesis.db"
COLLECT_INTERVAL=3600  # 1시간마다 수집
DEADMAN_DAYS=${GENESIS_DEADMAN_DAYS:-30}

# --- 초기화 ---
echo "[$(date -Iseconds)] daemon.sh started" >> "$LOG_DIR/daemon.log"

# DB 초기화 (최초 실행 시)
python3 /app/init_db.py "$DB_PATH"

# --- 메인 루프 ---
while true; do
    TIMESTAMP=$(date -Iseconds)

    # 1. Collector 모듈 순차 실행
    for collector in /app/collectors/*.py; do
        MODULE=$(basename "$collector" .py)
        echo "[$TIMESTAMP] Running collector: $MODULE" >> "$LOG_DIR/daemon.log"
        python3 "$collector" "$DB_PATH" 2>> "$LOG_DIR/${MODULE}.err" || true
    done

    # 2. PoW 블록 생성 (evidence 무결성 보장)
    python3 /app/pow_writer.py "$DB_PATH"

    # 3. Dead Man's Switch 확인
    python3 /app/deadman_check.py "$DB_PATH" "$DEADMAN_DAYS"

    # 4. 자기 상태 기록
    python3 /app/self_report.py "$DB_PATH"

    # 5. 대기
    sleep "$COLLECT_INTERVAL"
done
```

"**무한 루프.**" 김영훈이 말했다. "1시간마다 collector를 돌리고, PoW 블록을 쓰고, Dead Man's Switch를 확인하고, 자기 상태를 기록하고, 다시 잠든다."

"20년간 이 루프가 돌았어." 추박하가 말했다. "내가 자는 동안에도. 출장 중에도. 아플 때도."

---

가람은 수첩에 적었다.

> `[Agent Daemon — 자율 동작 구조]`
>
> `daemon.sh 구조:`
> `  무한 루프 (while true)`
> `  → collector 모듈 순차 실행`
> `  → PoW 블록 생성`
> `  → Dead Man's Switch 확인`
> `  → 자기 상태 기록`
> `  → sleep (기본 1시간)`
>
> `Collector 모듈 5종:`
> `  email_metadata: 메일 메타데이터 (본문 미수집)`
> `  network_flow: 비정상 네트워크 패턴`
> `  file_change: 학습 데이터 디렉토리 변경`
> `  schedule_task: crontab/timer 변경 (증거 인멸 시도 감지)`
> `  self_integrity: genesis 자체 변조 감지`
>
> `설계 원칙:`
> `  1. 자율성: 사람의 개입 없이 동작`
> `  2. 최소 수집: 필요한 메타데이터만 (본문, 내용 미수집)`
> `  3. 자기 무결성: 자기 자신도 감시`
> `  4. 안전 장치: Dead Man's Switch (30일)`
>
> `OpsClaw와의 관계:`
> `  genesis = OpsClaw의 원형`
> `  genesis의 PoW → OpsClaw의 PoW 체인`
> `  genesis의 collector → OpsClaw의 SubAgent + Skill`
> `  genesis의 daemon.sh → OpsClaw의 scheduler-worker`
> `  → "genesis가 없었으면 OpsClaw도 없었다"`
