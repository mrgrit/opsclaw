# Chapter 8: 문 앞의 발자국

---

금요일 오전. 송진호는 자기만의 세계에 빠져 있었다.

그의 모니터에는 세 개의 브라우저 창이 열려 있었다. VirusTotal, Shodan, 그리고 — 아무도 모르는 — 다크웹 포럼의 접속 화면.

"진호야, 뭐 찾았어?" 민재가 커피를 건네며 물었다.

진호가 모니터를 돌려 보여주었다.

"어제 SSH 공격자 45.33.32.156. 이 IP 자체는 Linode VPS야. 일시적으로 빌린 서버. 하지만 이 IP가 과거에 연결된 **다른 IP들**이 흥미로워."

---

### Passive DNS와 연관 IP 분석

진호는 Passive DNS 데이터를 추적했다.

```
45.33.32.156
  ├── scan.evil-test.xyz    (A레코드, 2026-03-15~현재)
  ├── c2.dropzone.click     (A레코드, 2026-02-20~03-10)
  └── update.nexus-cdn.com  (A레코드, 2026-01-05~01-20)
```

세 번째 도메인에서 진호가 멈추었다.

"**nexus-cdn.com**?" 민재가 화면을 들여다보았다. "이거 우리 회사 도메인 아니야?"

"아니." 진호가 고개를 저었다. "우리 회사는 **nexusai.co.kr**이야. nexus-cdn.com은 **유사 도메인**이야. 타이포스쿼팅."

"누가 왜 우리 회사와 비슷한 도메인을 만들었을까?"

진호의 눈이 반짝였다. "그게 문제지."

---

진호는 WHOIS 데이터를 조회했다.

```bash
whois nexus-cdn.com 2>/dev/null | grep -E "Registrant|Creation|Expiry"
```

```
Registrant Organization: Privacy Protect, LLC
Creation Date: 2025-12-15
Registry Expiry Date: 2026-12-14
```

"등록자 정보가 프라이버시 보호로 숨겨져 있어." 진호가 말했다. "2025년 12월에 등록. **4개월 전**이야."

"SSH 공격이 시작되기 전에 도메인을 먼저 확보한 거네." 민재가 말했다.

"맞아. 이건 **계획된 공격**이야. 스크립트 키디가 아니라, **사전에 인프라를 준비**한 공격자야."

---

진호는 다음 단계로 넘어갔다. nexus-cdn.com이 연결되었던 IP들을 역추적.

```
nexus-cdn.com → 과거 IP 기록:
  45.33.32.156  (2026-01-05~01-20)
  104.237.x.x   (2025-12-15~01-04)
  192.99.x.x    (2025-12-15, 최초 등록 시)
```

"192.99.x.x" 진호가 눈을 가늘게 떴다. "이건 OVH 캐나다. 러시아 APT 그룹이 자주 쓰는 호스팅이야."

"러시아?" 도윤이 놀라 물었다.

"단정할 수는 없어." 진호가 손을 흔들었다. "호스팅 업체만으로 국적을 특정할 수는 없어. 하지만 **TTP(전술·기법·절차) 패턴**을 보면..."

진호가 MITRE ATT&CK Navigator를 열었다.

```
관찰된 TTP:
- T1110.001 (Password Guessing) — SSH 브루트포스
- T1078 (Valid Accounts) — 탈취한 계정으로 접근
- T1136 (Create Account) — guest03 비인가 사용자 생성
- T1059.004 (Unix Shell) — SSH를 통한 셸 접근
- T1071.001 (Web Protocols) — nexus-cdn.com 유사 도메인 (C2 인프라)
```

"이 패턴은..." 진호가 ATT&CK 데이터베이스를 검색했다. "여러 APT 그룹과 겹치지만, nexus-cdn.com이라는 유사 도메인을 쓴다는 건 **우리 회사를 특정 목표로** 삼았다는 뜻이야."

---

김영훈이 다가왔다.

"진호, 요약해."

진호가 화이트보드에 적었다.

```
[위협 평가]

공격자 프로파일:
- 유형: 표적형 (Targeted) — 넥서스AI 유사 도메인 사전 준비
- 인프라: Linode + OVH 캐나다 VPS
- 기간: 최소 4개월 전부터 준비 (2025-12)
- TTP: SSH 브루트포스 → 계정 탈취 → 사용자 생성

평가:
- 스크립트 키디: ❌ (인프라 사전 준비, 유사 도메인)
- 사이버 범죄자: △ (가능하지만 동기 불명확)
- APT/국가 지원: △ (TTP 패턴 일부 일치, 확증 부족)
- 경쟁사: ⚠️ (넥서스AI 특정 목표, AI 기술 탈취 가능성)
```

관제실이 조용해졌다. "경쟁사"라는 단어가 공기를 무겁게 만들었다.

김영훈이 말했다. "경쟁사 가능성은... 서문기 상무에게 보고해야 하나?"

"아직." 추박하의 목소리가 들렸다. 모두가 뒤를 돌아보았다. 추박하는 만화책을 펼친 채로 말하고 있었다. "확증 없이 경쟁사를 거론하면 정치적 문제가 돼. 증거를 더 모아."

"하지만 팀장님, 유사 도메인은—"

"도메인 하나로 '경쟁사의 소행'이라고 단정하면, 우리가 바보가 되는 거야." 추박하가 만화책을 넘기며 말했다. "진호, nexus-cdn.com에 **어떤 콘텐츠가 호스팅**되어 있었는지 확인해. Wayback Machine이든 캐시든."

"네."

"그리고 영훈아."

"네, 팀장님."

"web 서버의 auth.log에서 45.33.32.156이 **어떤 사용자명으로** 로그인을 시도했는지 봐. '1'이라는 비밀번호를 알고 있었다면, **사용자명도 알고 있었을** 가능성이 높아."

김영훈의 표정이 변했다. 그 질문을 놓치고 있었다.

---

```bash
curl -s -X POST "http://localhost:8000/projects/$PID/dispatch" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: opsclaw-api-key-2026" \
  -d '{"command":"grep \"45.33.32.156\" /var/log/auth.log 2>/dev/null | grep \"Failed\\|Accepted\" | head -20","subagent_url":"http://10.20.30.80:8002"}'
```

결과:

```
Mar 28 15:18:33 web sshd[78201]: Failed password for web from 45.33.32.156 port 49821 ssh2
Mar 28 15:18:35 web sshd[78201]: Failed password for web from 45.33.32.156 port 49821 ssh2
Mar 28 15:18:37 web sshd[78201]: Failed password for web from 45.33.32.156 port 49821 ssh2
...
Mar 28 15:21:01 web sshd[78234]: Accepted password for web from 45.33.32.156 port 49823 ssh2
```

모든 시도가 **`web`** 사용자명이었다.

"**사용자명을 알고 있었어.**" 김영훈이 말했다. "23번 모두 'web' 계정으로만 시도했어. 다른 사용자명은 시도하지 않았어."

"그건..." 정규가 말했다. "우리 서버의 사용자명을 **이미 알고 있었다**는 뜻이잖아요."

"맞아." 추박하가 만화책을 덮었다. 두 번째로 진지해지는 순간이었다.

"이건 **내부 정보가 유출**되었을 가능성이 있어."

관제실이 얼어붙었다.

---

가람은 그 순간의 공기를 잊을 수 없었다.

*내부 정보 유출.*

그 말이 의미하는 것은 — 넥서스AI 안에 **누군가가 공격자에게 정보를 건넸다**는 것이었다.

가람은 수첩에 적었다.

> `공격자는 'web' 사용자명을 알고 있었다.`
> `넥서스AI 내부에 정보 유출자가 있을 가능성.`
> `유사 도메인 nexus-cdn.com — 4개월 전부터 준비된 표적 공격.`
> `추팀장이 "증거를 더 모아라"고 함. 경쟁사 단정 금지.`

그리고 수첩 한 페이지를 넘기고 적었다.

> `질문: 추팀장은 이 모든 것을 이미 알고 있는 걸까?`
> `그가 새벽마다 genesis에 기록하는 것은... 이런 내용?`
