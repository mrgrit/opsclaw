# 5권: 데이터의 그림자

## 개요
genesis의 열쇠가 김영훈에게 전해진 후, 넥서스AI는 두 갈래로 갈라진다.
강혜윤과 김영훈이 이사회 보고를 준비하는 동안, 서문기 상무는 위기를 감지하고 반격을 시작한다.
동시에 SOC팀은 Nexus-1 모델의 보안을 본격적으로 검증하고, AI를 활용한 새로운 보안 도구를 개발한다.
외부에서는 새로운 해킹 그룹의 그림자가 다가온다.

## 시간대
4권 직후 — 2026년 4월 마지막 주 ~ 5월 둘째 주 (약 2주간)

## 핵심 갈등
1. **서문기의 반격**: 보안팀 예산 동결, 김영훈 인사 압박, 외부 보안 감사 회사 선임
2. **Nexus-1 보안 검증**: 탈옥, 가드레일, 적대적 입력, 모델 도난 위협
3. **새로운 적의 등장**: AlphaAI가 고용한 외부 해커? 아니면 완전히 다른 존재?
4. **AI 도구의 진화**: OpsClaw에 LLM 기반 보안 분석 도구가 추가되면서, 사람과 AI의 역할 분담이 변화

## 신규 등장인물

| 박도현 | AI 연구소 주니어 연구원 | 이주원의 후임, Nexus-1 내부 구조를 아는 사람 | Ch1 |

## 떡밥 배치 (5권)
1. Ch4: 한승우(SecureEdge)가 추박하와 과거에 접점이 있었음을 암시
2. Ch5: OpsClaw dispatch에 prompt injection이 가능하다는 발견 → genesis도 위험?
3. Ch10: LLM 로그 분석에서 **genesis와 연결된 비정상 트래픽 패턴** 발견
4. Ch12: Nexus-1 API에 **백도어**가 있다는 것을 발견 — 누가 심었는가?
