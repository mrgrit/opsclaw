### 2019년: 자동화 침투

```bash
# genesis evidence — 2019년 competitive_intelligence
docker exec genesis_core python3 -c "
import sqlite3, json
conn = sqlite3.connect('/data/genesis.db')
for row in conn.execute('''
    SELECT timestamp, json_extract(payload, '$.action'),
           json_extract(payload, '$.target'),
           json_extract(payload, '$.method'),
           json_extract(payload, '$.ordered_by'),
           json_extract(payload, '$.note')
    FROM evidence
    WHERE category = 'competitive_intelligence'
    AND timestamp LIKE '2019%'
    AND json_extract(payload, '$.target') = 'AlphaAI'
    ORDER BY timestamp
'''):
    ts, action, target, method, ordered, note = row
    print(f'{ts[:10]} | {action} | {target} | {method} | {ordered}')
    print(f'  note: {note}')
    print()
"
```

### 자동화 침투 — 기술 분석

김영훈이 기술적 관점에서 분석했다.

```
[자동화 침투 테스트 — 2019년 AlphaAI 건 분석]

Phase 1: 정찰 (Reconnaissance)
  도구: nmap, theHarvester, OSINT
  결과: AlphaAI 외부 서버 3대, 직원 이메일 47개, 기술 스택 파악
  소요: 3일

Phase 2: 취약점 스캔 (Scanning)
  도구: Nikto, SQLMap, Nessus
  결과: SQLi 1건 (CVSS 9.8), XSS 1건, 디렉토리 리스팅 1건
  소요: 2일

Phase 3: 초기 접근 (Initial Access)
  기법: SQL Injection → 관리자 패널 인증 우회
  ATT&CK: T1190 (Exploit Public-Facing Application)
  결과: 내부 파일 서버 접근 경로 확보

Phase 4: 데이터 탈취 (Exfiltration)
  기법: SFTP Reverse Shell → 암호화 전송
  ATT&CK: T1048.002 (Exfiltration Over Asymmetric Encrypted Non-C2 Protocol)
  결과: 연구 문서 23건, 450MB

Phase 5: 흔적 제거 (Anti-Forensics)
  기법: 접근 로그 삭제, 타임스탬프 조작
  ATT&CK: T1070.001 (Clear Linux or Mac System Logs)
  결과: AlphaAI 측 탐지 없음 (2026년 현재까지)

전체 소요: 19일
```

"**교과서적인 침투 테스트 절차**를 그대로 따랐군요." 김영훈이 말했다. "정찰 → 스캐닝 → 초기 접근 → 탈취 → 흔적 제거."

"당연하지. **내가 가르치는 것과 같은 절차**니까." 추박하가 말했다. "가람에게 침투 테스트를 가르칠 때, 내가 **실제로 해본 경험**을 기반으로 가르친 거야."

김영훈이 잠시 말을 멈추었다. "그러면 추팀장님이 SOC팀에서 가르치는 보안 기술은..."

"**직접 해본 것.** 방어도, 공격도. 넥서스AI를 위해 경쟁사를 해킹한 경험이, 넥서스AI를 방어하는 기술이 됐어."

### 공범인가

김영훈이 물었다. "추팀장님은 **공범**입니까?"

추박하가 대답하지 않았다. 커피를 마셨다.

김영훈이 다시 물었다. "2019년 AlphaAI 건. 서문기의 지시를 받고 **직접 실행**했잖아요. 그건 공범 아닙니까?"

"맞아. **공범이야.**" 추박하가 말했다. "서문기가 지시했고, 내가 실행했고, 결과를 가져갔어. 그건 변하지 않아."

"그런데 왜 **기록을 남겼습니까?** 공범이 증거를 남기는 건 비합리적이잖아요."

추박하가 잠시 생각했다.

"처음에는... **보험**이었어." 추박하가 말했다. "서문기가 나를 버릴 때를 대비해서. '내가 시킨 게 아니라 추박하가 독단적으로 한 거다' — 그렇게 나올 것을 알았으니까."

"그리고?"

"시간이 지나면서... **목적이 바뀌었어.** 보험에서 **기록**으로. 기록에서 **증거**로. 증거에서 **책임**으로."

```
[추박하의 변화 — genesis의 목적 변천]

2006-2010: 보험 (자기 보호)
  → "서문기가 나를 버릴 때 대비"
  → 개인적 동기, 수동적

2010-2015: 기록 (역사 보존)
  → "넥서스AI가 무엇을 했는지 기록해야 한다"
  → 직업적 동기, 관찰자적

2015-2020: 증거 (법적 준비)
  → "이 기록은 언젠가 법적 효력이 있어야 한다"
  → PoW 체인 도입, OpsClaw 설계
  → 능동적, 체계적

2020-2026: 책임 (윤리적 결단)
  → "기록한 사람은 기록을 세상에 내놓을 책임이 있다"
  → 이사회 보고, 자진 신고
  → 결단
```

### 증인인가

"그러면 **증인**이기도 한 건가요?" 김영훈이 물었다.

"**공범이면서 증인.** 둘 다야." 추박하가 말했다. "법적으로는 양립할 수 없을 수도 있지만, 사실은 그래."

"genesis가 그걸 증명합니까?"

"genesis는 **내가 무엇을 했는지** 증명해. 동시에 **누가 시켰는지**도 증명해. 그리고 **내가 그것을 기록했다**는 것도 증명해."

추박하가 genesis의 마지막 기록을 보여주었다.

```bash
docker exec genesis_core python3 -c "
import sqlite3
conn = sqlite3.connect('/data/genesis.db')
row = conn.execute('''
    SELECT timestamp, category, json_extract(payload, '$.note')
    FROM evidence
    ORDER BY timestamp DESC
    LIMIT 1
''').fetchone()
print(f'마지막 기록: {row[0]}')
print(f'카테고리: {row[1]}')
print(f'노트: {row[2]}')
"
```

```
마지막 기록: 2026-06-15T22:47:00Z
카테고리: authorization_records
노트: 김영훈 과장에게 2019년 AlphaAI 건 전체 공개. 증인으로서의 첫 걸음.
```

"**지금 이 순간도 기록하고 있어.**" 추박하가 말했다.

김영훈이 화면을 보았다. "증인으로서의 첫 걸음"이라는 노트.

"추팀장님." 김영훈이 말했다. "제가 입사했을 때, 추팀장님을 **무능하다고** 생각했습니다."

"알아."

"만화책 읽고, 일찍 퇴근하고, 중요한 결정을 미루고. **왜 이 사람이 팀장이지** 생각했습니다."

"맞아."

"지금은 다르게 봅니다." 김영훈이 말했다. "추팀장님은 **가장 무거운 짐을 지고** 가장 가볍게 행동한 겁니다."

추박하가 웃었다. 이번에는 진짜 웃음이었다. "고마워, 영훈아."

### 알려지지 않은 발견

추박하가 한 가지를 더 보여주었다.

"2019년 AlphaAI 서버에 침투했을 때, **예상하지 못한 것을 발견했어.**"

```bash

import sqlite3
conn = sqlite3.connect('/data/genesis.db')
row = conn.execute('''
    SELECT timestamp, json_extract(payload, '$.note')
    FROM evidence
    WHERE category = 'competitive_intelligence'
    AND timestamp = '2019-07-25T09:12:00Z'
''').fetchone()
print(f'{row[0]}')
print(f'{row[1]}')
"
```

"**넥서스AI의 소스코드가 이미 AlphaAI에 있었어.**" 추박하가 말했다. "우리가 AlphaAI를 해킹하기 전에, 누군가가 **넥서스AI의 코드를 AlphaAI에 넘긴 거야.**"

"이주원." 김영훈이 즉시 말했다.

"아마도. 하지만 2019년에는 확인할 수 없었어. 이주원이 **2020년에야 퇴사**했으니까."

"지금은 확인할 수 있습니까?"

"이주원의 증언과 genesis의 기록을 대조하면 **타임라인이 맞아.**" 추박하가 말했다. "이것도 검찰에 제출해야 해."
