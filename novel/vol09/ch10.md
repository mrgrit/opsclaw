목요일 오전 8시. 넥서스AI SOC 관제실.

강혜윤이 관제실에 들어왔다. 평소와 달리 두 사람이 동행하고 있었다.

"검찰 **디지털포렌식팀** 소속 수사관 두 분입니다." 강혜윤이 말했다. "genesis evidence의 무결성 검증을 위해 오셨습니다."

관제실이 얼어붙었다.

민재가 커피를 쏟았다. 이번에는 아무도 웃지 않았다.

### 검찰 방문

수사관 장현우. 40대 초반. 디지털포렌식 전문가. 말이 적고 정확했다.

수사관 김소연. 30대 중반. 데이터 분석가 출신. 노트북에 Autopsy, FTK가 설치되어 있었다.

"추박하 팀장님이시죠?" 장현우가 말했다.

"네." 추박하가 일어섰다. 만화책도, 커피잔도 없었다.

"genesis 프로젝트의 **PoW 체인 무결성을 독립 검증**하러 왔습니다. 넥서스AI가 자진 신고한 건에 대한 보강 증거로 사용할 예정입니다."

"알겠습니다." 추박하가 말했다.

추박하가 genesis에 접근하는 과정을 수사관에게 보여주었다. 김영훈이 기술적 설명을 담당했다.

### PoW 체인 독립 검증

장현우 수사관이 자신의 노트북에서 직접 검증을 수행했다. 넥서스AI의 도구를 사용하지 않고, **독립적인 검증 스크립트**를 실행했다.

```bash
# 수사관의 독립 검증 스크립트 (넥서스AI 코드와 무관)
python3 verify_pow_chain.py \
  --db /data/genesis.db \
  --table evidence \
  --hash-column pow_hash \
  --prev-column prev_hash \
  --difficulty 4

# 출력:
# Verifying PoW chain for genesis.db...
# Total blocks: 40,132
# Chain start: 2006-03-15T09:23:17Z (block #1)
# Chain end:   2026-06-18T08:55:00Z (block #40132)
#
# Verification:
#   Valid blocks: 40,132
#   Invalid blocks: 0
#   Orphan blocks: 0
#   Chain integrity: VALID
#   Hash difficulty: 4 (all blocks meet criteria)
#
# Conclusion: The PoW chain is cryptographically intact.
# No evidence of tampering detected.
```

"**40,132건. 전부 유효합니다.**" 장현우가 말했다.

김소연이 데이터를 분석했다.

```
[genesis evidence 검증 결과 — 검찰 독립 검증]

총 레코드: 40,132건
기간: 2006-03-15 ~ 2026-06-18 (20년 3개월)
체인 무결성: VALID (변조 없음)

카테고리별:
  customer_data_ops:     12,567건 (31.3%)
  system_maintenance:     9,342건 (23.3%)
  competitive_intelligence: 8,234건 (20.5%)
  internal_audit:         6,789건 (16.9%)
  authorization_records:  3,200건 (8.0%)

연도별 분포:
  2006-2010:  3,421건 (수동 기록, SQLite 직접 입력)
  2011-2015:  7,892건 (스크립트 자동화 도입)
  2016-2020: 12,345건 (OpsClaw PoW 체인 마이그레이션)
  2021-2026: 16,474건 (Agent Daemon 자동 기록)

특이 사항:
  - 2006-2015 레코드는 원래 SQLite 직접 입력 → 2020년 PoW 체인으로 일괄 마이그레이션
  - 마이그레이션 전/후 해시가 모두 보존되어 있어, 원본 무결성 검증 가능
  - authorization_records 3,200건에 음성 파일 참조 포함 (별도 암호화 볼륨)
```

"마이그레이션 전 원본도 보존되어 있나요?" 김소연이 물었다.

"네." 추박하가 말했다. "**genesis.db.legacy**라는 파일에 2006-2015년 원본이 그대로 있습니다. PoW 체인 마이그레이션 시 원본 해시를 메타데이터로 포함시켰습니다."

"철저하시네요." 장현우가 말했다. 감탄도 판단도 아닌, 사실 확인의 어조.

### THE BIG REVEAL

검찰의 검증이 끝난 후. 오후 3시.

강혜윤이 관제실에 돌아왔다.

"검찰이 genesis evidence를 **공식 증거로 채택**했습니다."

관제실이 웅성거렸다.

"그리고..." 강혜윤이 말을 이었다. "오늘 오후 4시에 **검찰 브리핑**이 있습니다. 언론에 자진 신고 건이 공개됩니다."

"언론이요?" 민재가 놀랐다.

"넥서스AI가 **학습 데이터 불법 수집 및 경쟁사 해킹**에 대해 자진 신고했다는 내용입니다. genesis의 존재도 보도됩니다."

추박하가 가만히 앉아 있었다. 20년간 숨겨왔던 것이 세상에 나오는 순간이었다.

```
[YTN 속보 — 2026.06.18 17:00]
```

관제실 모니터에 뉴스가 떠 있었다. 아무도 말하지 않았다.

"추모씨(46)." 도윤이 읽었다. "추팀장님이잖아요."

"그래." 추박하가 말했다.

"추팀장님이 **체포되는 건 아니죠?**" 민재가 물었다. 목소리가 떨렸다.

"자진 신고에 협조하고 있으니까 **영장은 아직 없어.**" 강혜윤이 말했다. "하지만 조사는 받아야 해."

### OpsClaw = 블랙박스

뉴스가 확산되면서, 넥서스AI 전체가 흔들렸다. 하지만 관제실에서 김영훈은 다른 것을 보고 있었다.

"OpsClaw가 **회사의 블랙박스**라고 보도됐어." 김영훈이 말했다.

"비행기의 블랙박스처럼." 가람이 말했다.

"맞아." 김영훈이 말했다. "비행기 블랙박스는 **사고 원인을 밝히기 위해** 존재해. 사고가 나기 전에는 아무도 블랙박스를 신경 쓰지 않지. 하지만 사고가 나면 — **가장 중요한 것이 돼.**"

```
[OpsClaw = 블랙박스]

비행기 블랙박스:
  - Flight Data Recorder (FDR): 비행 데이터 기록
  - Cockpit Voice Recorder (CVR): 조종실 음성 기록
  - 사고 후 원인 규명에 결정적 역할
  - 위변조 방지 설계 (내화, 내수, 충격 방호)

OpsClaw:
  - PoW 체인: 모든 작업 데이터 기록
  - genesis: 20년간의 authorization records (지시-실행-결과)
  - 사건 후 진실 규명에 결정적 역할
  - 위변조 방지 설계 (PoW 해시 체인, Docker 격리)

공통점:
  1. 평소에는 눈에 띄지 않음
  2. 위변조가 불가능하도록 설계됨
  3. 사건 발생 시 가장 중요한 증거가 됨
  4. 기록의 목적은 "진실의 보존"

차이점:
  블랙박스 — 기관이 설치하고, 규정에 따라 운영
  genesis — 개인이 설치하고, 스스로의 윤리에 따라 운영
  → 추박하의 genesis는 "1인 블랙박스"
```

"1인 블랙박스." 가람이 되뇌었다.

"20년간 혼자서 블랙박스를 운영한 사람." 정규가 말했다. "그게 추팀장이야."

### 추박하의 전화

저녁 7시. 관제실에 추박하만 남아 있었다.

전화벨이 울렸다. 추박하가 받았다.

"... 네, 어머니."

...

"뉴스에요? 아, 네. 그게 저예요."

...

"아니요, 체포된 건 아니에요. 조사만..."

...

"걱정하지 마세요. 제가 **잘못한 건 맞지만**, 기록을 남겼으니까 — 그게 도움이 될 거예요."

...

"네. 밥은 먹었어요."

전화를 끊고, 추박하가 천장을 올려다보았다.

가람이 관제실 입구에 서 있었다. 잊고 간 수첩을 가지러 온 것이었다. 추박하의 전화를 들은 것은 — 의도하지 않았다.

추박하가 가람을 보았다.

"... 들었어?"

"죄송합니다." 가람이 고개를 숙였다.

"괜찮아." 추박하가 말했다. "어머니는 내가 **회사에서 나쁜 짓을 하다 걸린 줄** 알아. 틀린 건 아니지만, 전부도 아니야."

가람은 수첩을 집어 들었다. 그리고 물었다.

"추팀장님. 후회하세요?"

추박하가 잠시 생각했다.

"genesis를 만든 건 **후회하지 않아.** 경쟁사를 해킹한 건 **후회해.** 그리고 20년을 기다린 건 — **후회하기도 하고, 아니기도 해.** 더 일찍 세상에 내놓았으면 피해가 줄었을 수도 있어. 하지만 더 일찍 내놓았으면 기록이 **불완전**했겠지."

"완전한 기록을 위해 **20년을 참았다**는 건가요?"

"참은 건지, **미룬 건지** 나도 모르겠어." 추박하가 말했다. "하지만 지금 이 순간, genesis에는 40,132건의 진실이 있어. 그것만큼은 확실해."
