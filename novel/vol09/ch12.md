```yaml

steps:
  - order: 1

    tool: run_command
    params:
      command: "curl -s http://localhost:8000/pow/verify?agent_id=http://localhost:8002"

  - order: 2
    name: evidence_export
    description: "genesis evidence 전체를 포렌식 표준 형식(E01)으로 내보내기"

    params:

    risk_level: medium

  - order: 3
    name: hash_manifest
    description: "내보낸 파일의 SHA-256 해시 매니페스트 생성"
    tool: run_command
    params:
      command: "sha256sum /data/export/genesis_evidence_20260620.e01 > /data/export/manifest.sha256"
    risk_level: low

  - order: 4
    name: encrypt_package
    description: "검찰 공개키로 암호화 패키징"
    tool: run_command
    params:
      command: "gpg --encrypt --recipient prosecutor-forensics@spo.go.kr /data/export/genesis_evidence_20260620.e01"
    risk_level: high

  - order: 5
    name: transfer_verify

    params:

    risk_level: critical

  - order: 6
    name: completion_report
    description: "genesis 최종 보고서 생성"
    tool: run_command
    params:
      command: "curl -X POST http://localhost:8000/projects/prj_000000000000/completion-report -H 'Content-Type: application/json' -H 'X-API-Key: $OPSCLAW_API_KEY' -d '{\"summary\":\"genesis evidence 전체 검찰 제출 완료\",\"outcome\":\"success\",\"work_details\":[\"40132건 evidence\",\"PoW 체인 검증 통과\",\"E01 포맷 내보내기\",\"GPG 암호화\",\"검찰 전송\"]}'"
```

### Playbook 개념 복습

가람이 물었다. "Playbook이 정확히 뭔지 다시 설명해 주실 수 있어요? 처음 들었을 때보다 **의미가 다르게 느껴져서.**"

김영훈이 설명했다.

```
[OpsClaw Playbook — 심화]

Playbook이란?
  사전에 정의된 작업 절차를 자동화한 것.
  SOAR(Security Orchestration, Automation, and Response)의 핵심 구성 요소.

구조:
  - steps: 순차적으로 실행되는 작업 목록
  - 각 step: tool(도구), params(매개변수), risk_level(위험도)
  - execute-plan API로 실행

Playbook vs 수동 실행:
  수동: 사람이 각 단계를 하나씩 실행 → 실수 가능, 느림
  Playbook: 정의된 절차를 자동 실행 → 정확, 빠름, 기록 자동화

PoW와의 연동:
  - Playbook의 각 step 실행 결과가 PoW 블록으로 기록
  - 전체 Playbook 완료 시 completion-report 생성
  - 누가 무엇을 언제 실행했는지 위변조 불가능하게 보존

RL(강화학습)과의 연동:
  - 각 task의 성공/실패가 reward로 기록
  - RL이 최적 risk_level을 학습
  - 다음 실행 시 RL 추천에 따라 risk_level 자동 조정
```

"처음에 Playbook은 **효율성을 위한 도구**라고 생각했어요." 가람이 말했다. "하지만 지금은 — **책임을 증명하는 도구**이기도 하네요."

"맞아." 추박하가 말했다. "genesis의 마지막 Playbook은 **내 책임의 마지막 증명**이야."
