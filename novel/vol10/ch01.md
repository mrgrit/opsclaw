# 10권 제1장: 증거 제출

## 강의 연계: C3W14(심화) — 보고서 작성

---

서울중앙지방검찰청 12층 특수수사과.

강혜윤은 복도의 형광등 아래에서 은색 하드케이스를 무릎 위에 올려두고 앉아 있었다. 케이스 안에는 외장 SSD 두 대, 해시값이 인쇄된 봉인 라벨, 그리고 182페이지짜리 보고서가 들어 있었다. 넥서스AI 내부 데이터 은폐 사건의 모든 증거—genesis라는 코드명으로 수집된 모든 것.

옆에 앉은 가람이 조용히 물었다.

"혜윤 선배, 긴장돼요?"

"아니."

거짓말이었다. 강혜윤의 손끝이 미세하게 떨리고 있었다. 하지만 그녀는 포렌식 분석관이었고, 자신이 만든 보고서를 믿었다.

---

세 시간 전, SOC 사무실.

강혜윤은 최종 보고서를 프린트하기 전에 팀원들 앞에서 구조를 한 번 더 설명했다.

"보고서는 총 다섯 파트야."

그녀가 화이트보드에 적었다.

```
1. Executive Summary — 사건 개요 (1페이지)
2. 타임라인 — 발생~탐지~대응~현재 (15페이지)
3. 기술적 분석 — 포렌식 증거, 로그, 해시 (80페이지)
4. 영향 분석 — 피해 범위, 유출 데이터 목록 (40페이지)
5. 부록 — 원본 로그, 스크린샷, 해시 검증 (46페이지)
```

가람이 노트북에 적으며 물었다.

"Executive Summary가 왜 첫 번째예요? 기술적 분석이 핵심 아닌가요?"

"검사가 기술자가 아니니까."

강혜윤이 단호하게 말했다.

"보고서의 첫 페이지는 읽는 사람이 결정해. 이 보고서를 읽는 건 검사야. 검사는 '무슨 일이 있었고, 누가 했고, 증거가 뭔지'를 알고 싶은 거지, TCP 패킷 헤더의 바이트 오프셋이 궁금한 게 아니야."

김영훈이 고개를 끄덕였다.

"그래서 Executive Summary에는 뭘 넣었어?"

"세 문장이야."

강혜윤이 화면을 전환했다.

```
넥서스AI는 2019년부터 2026년까지 고객 개인정보 유출 사실을 인지하면서도
내부적으로 은폐하고, 관련 로그를 조작하였습니다.
본 보고서는 해당 사실을 입증하는 기술적 증거와 타임라인을 제시합니다.
```

"이 세 문장이 182페이지의 요약이야. 검사가 이걸 읽고 '더 보겠다'고 결정하면, 그때 타임라인으로 넘어가는 거지."

---

조은별이 타임라인 파트를 가리켰다.

"여기, 2024년 3월 14일 로그 삭제 시점이요. 이 부분의 근거가 좀 약하지 않아요? OpsClaw 로그만으로 충분한가요?"

강혜윤이 고개를 저었다.

"아니, 그래서 교차 검증을 세 겹으로 했어."

```
증거 교차 검증 (Cross-Validation):
1. OpsClaw evidence_pack — 타임스탬프 + 해시체인 (변조 불가)
2. Wazuh SIEM 로그 — 독립적 로그 수집 시스템의 동일 시점 기록
3. AWS CloudTrail — 삭제 API 호출 기록 (서문기의 IAM 계정)
```

"세 개의 독립적인 소스가 같은 시점, 같은 행위를 가리키면—그건 우연이 아니야. 법정에서도 부정하기 어려워."

가람이 적었다.

```
[가람의 노트]
보고서 작성 핵심 원칙:
- 독자를 먼저 파악할 것 (기술자 vs 비기술자)
- Executive Summary는 3문장 이내
- 증거는 반드시 교차 검증 (최소 2개 독립 소스)
- 타임라인은 시간순, 모든 항목에 출처 명시
- 해시값으로 원본 무결성 보장
```

---

정규가 조심스럽게 손을 들었다.

"선배, 해시 검증 부분이요. 보고서에 SHA-256 해시를 넣었는데, 검사가 이게 뭔지 모르면 어떻게 해요?"

강혜윤이 미소 지었다. 좋은 질문이었다.

"그래서 부록에 '비전문가를 위한 해시 설명'을 한 페이지 넣었어."

```
[부록 A-1: 디지털 지문(해시)이란?]

문서에 지문이 있듯, 디지털 파일에도 고유한 지문이 있습니다.
SHA-256이라는 알고리즘은 파일의 내용을 256비트 문자열로 변환합니다.
파일이 1비트라도 바뀌면 이 문자열이 완전히 달라집니다.

예시:
  원본 파일 해시: a3f2b8c9d1e4...
  1글자 변조 후:  7f1e9a2c8b5d... (완전히 다름)

따라서 해시값이 일치하면, 해당 파일은 원본과 동일함이 수학적으로 보장됩니다.
```

"기술 보고서에서 가장 중요한 건 정확성이야. 두 번째로 중요한 건 '읽는 사람이 이해할 수 있는가'야. 이 두 가지가 충돌하면 안 돼. 정확하면서도 이해할 수 있어야 해."

---

김영훈이 최종 점검 목록을 읽었다.

"체크리스트 마지막으로 한 번 가자."

```
최종 제출 전 체크리스트:
[x] 모든 증거 파일의 SHA-256 해시 일치 확인
[x] 타임라인 내 모든 항목에 출처 페이지 번호 기재
[x] Executive Summary 검토 (비기술 독자 기준)
[x] 개인정보 마스킹 확인 (증인 외 개인정보 비식별화)
[x] 봉인 라벨 부착 및 이중 봉인
[x] 사본 3부 준비 (검찰 2부, 자체 보관 1부)
```

"전부 확인했어요."

강혜윤이 하드케이스를 닫았다. 잠금장치가 찰칵 소리를 냈다.

"가자."

---

다시 검찰청 복도.

문이 열리고, 수사관이 나왔다.

"강혜윤 분석관님, 들어오시죠."

강혜윤이 일어섰다. 가람도 따라 일어났지만, 수사관이 손을 들었다.

"한가람 씨는 잠시 대기해 주세요. 먼저 강 분석관과 이야기하겠습니다."

가람이 고개를 끄덕이고 다시 앉았다.

강혜윤이 돌아보지 않고 걸어 들어갔다. 하드케이스를 든 오른손이 이제는 떨리지 않았다.

---

수사관 앞에서 강혜윤은 케이스를 열었다.

"넥서스AI 내부 데이터 은폐 사건 관련 기술 분석 보고서입니다. SSD 두 대에 전체 evidence_pack이 담겨 있고, 봉인 라벨에 해시값이 인쇄되어 있습니다."

수사관이 보고서를 펼쳤다.

"이 보고서의 신뢰성은 어떻게 보장합니까?"

"세 가지로 보장합니다."

강혜윤의 목소리가 차분하고 명확했다.

"첫째, OpsClaw 시스템의 해시 체인. 모든 증거는 수집 시점에 SHA-256으로 해시되고, 이전 해시와 연결됩니다. 블록체인과 같은 원리입니다. 중간에 하나라도 조작하면 체인 전체가 깨집니다."

"둘째, 독립적 SIEM 로그. Wazuh라는 별도의 보안 모니터링 시스템이 동일한 시점의 로그를 독립적으로 수집했습니다. OpsClaw 로그와 Wazuh 로그가 교차 검증됩니다."

"셋째, AWS CloudTrail. 클라우드 인프라 제공자가 독립적으로 기록한 API 호출 로그입니다. 삭제 행위의 주체, 시점, IP가 모두 기록되어 있습니다."

수사관이 고개를 끄덕였다.

"182페이지라... 시간이 좀 걸리겠군요."

"Executive Summary를 먼저 읽어주시면 됩니다. 첫 페이지입니다."

---

40분 후, 가람이 불려 들어갔다.

수사관이 물었다.

"한가람 씨, OpsClaw라는 시스템의 개발에 참여하셨습니까?"

"네. SOC팀 분석가로서 시스템 운영에 참여했고, 일부 모니터링 룰을 작성했습니다."

"이 시스템이 수집한 증거를 조작할 가능성이 있습니까?"

가람은 잠시 생각했다. 그리고 정확하게 대답했다.

"OpsClaw의 evidence_pack은 해시 체인으로 보호됩니다. 체인의 어떤 블록이든 변조하면, 그 이후 모든 블록의 해시가 불일치합니다. 검증 API를 호출하면 즉시 탐지됩니다."

가람은 화면을 보여주었다.

```bash
curl http://localhost:8000/pow/verify?agent_id=http://localhost:8002
# {"valid": true, "blocks": 847, "orphans": 0, "tampered": []}
```

"847개의 블록이 모두 유효하고, 변조된 블록은 0개입니다."

수사관이 기록을 적었다.

---

검찰청을 나올 때, 서울의 하늘은 흐렸다.

강혜윤이 처음으로 한숨을 내쉬었다.

"끝났다."

"끝난 건가요?"

가람이 물었다.

"보고서 제출은 끝났어. 하지만 이건 시작이야. 이제부터 법정에서 이 보고서가 싸워야 해."

강혜윤이 하늘을 올려다보았다.

"우리가 할 수 있는 건 했어. 정확하게, 빠짐없이, 검증 가능하게."

가람이 노트북을 꺼내 마지막 줄을 적었다.

```
[가람의 노트]
보고서는 무기다.
정확한 보고서는 진실의 무기다.
오늘 우리는 그 무기를 건넸다.
```

택시를 타고 돌아가는 길, 강혜윤이 창밖을 보며 말했다.

"가람아."

"네?"

"보고서 작성법 가르쳐줄까? 제대로."

가람이 고개를 끄덕였다.

"네. 부탁합니다."

강혜윤이 작게 웃었다. 전쟁이 끝난 것은 아니었다. 하지만 탄약은 충분히 준비해 두었다.

---

*— 10권 1장 끝 —*
