# Chapter 1: 돌아온 사람

---

4월 셋째 주 월요일.

넥서스AI 관제실의 아침은 늘 비슷했다. 민재가 커피를 망치고, 정규가 모니터에 빠지고, 도윤이 야간 인수인계를 끝내고 비틀거리며 나가고.

"도윤아, 얼굴이 왜 그래?" 가람이 물었다.

"밤새 경보가 137건이었어." 도윤이 눈을 비비며 말했다. "정규 선배가 만든 새 탐지 룰이... 좀 **공격적**이야."

"오탐이 많았어?" 정규가 모니터에서 눈을 떼지 않고 물었다.

"137건 중 134건이 오탐이었어요." 도윤이 억울한 표정을 지었다.

"그럼 3건은 **진짜**잖아." 정규가 담담하게 말했다. "134번의 거짓을 확인하는 건, 3번의 진짜를 놓치지 않기 위해서야."

도윤이 한숨을 쉬었다. "그래도 134번은 좀..."

"분석가의 삶이야." 정규가 로그를 스크롤했다. "익숙해져."

지후가 지나가며 도윤의 어깨를 두드렸다. "나도 신입 때 밤새 로그 보다가 모니터에 얼굴 찍은 적 있어. **키보드 자국**이 뺨에 찍혀서 출근."

"진짜요?"

"진짜. 팀장님이 와서 사진 찍었어." 지후가 씩 웃었다. "아직 어딘가에 있을 거야."

관제실에 웃음이 퍼졌다. 추박하의 자리에서 "크크" 하는 소리가 들렸지만, 만화책에 가려 표정은 안 보였다.

---

그때 민재의 전화가 울렸다.

"여보세요? 네. 네? ... **뭐라고요?**"

민재의 얼굴이 변했다. 관제실이 조용해졌다.

민재가 전화를 끊고 말했다.

"이주원이 나타났대."

---

이주원은 사라진 지 10일 만에 **서울 강남 경찰서에 자진 출석**했다.

소식은 진수현 실장(인사실)을 통해 전해졌다.

"이주원이 **변호사를 대동**하고 출석했어요. 그리고 한 마디 했답니다."

"뭐래요?"

진수현이 잠시 멈추었다.

"'나는 산업 스파이가 아닙니다. **내부 고발자**입니다.'"

---

관제실이 술렁였다.

"내부 고발자?" 진호가 말했다. "그게 무슨—"

"이주원의 변호사가 경찰에 **진술서**를 제출했어요." 진수현이 서류를 읽었다. "요약하면—"

```
[이주원 진술서 요약]

1. 본인은 넥서스AI AI 연구소 모델 학습팀 선임연구원으로,
   Nexus-1 학습 데이터의 수집 및 전처리를 담당했다.

2. 학습 데이터의 일부가 경쟁사(AlphaAI)의 내부 자료,
   공개되지 않은 학술 논문, 고객사 개인정보를 포함하고 있음을
   업무 중 발견하였다.

3. 이 사실을 서문기 상무에게 보고하였으나,
   "릴리즈 일정에 영향을 주지 마라"는 지시를 받았다.

4. 내부 고발 채널(윤리경영실)에도 제보하였으나,
   2주 후 "조사 결과 문제없음"이라는 회신을 받았다.

5. 합법적 채널이 막혔다고 판단하여,
   증거(학습 데이터 목록, 아키텍처 문서, 모델 가중치)를
   외부에 확보한 후 자진 출석하였다.

6. 확보한 데이터는 변호사에게 위탁하였으며,
   필요 시 관계 기관(KISA, 개인정보보호위원회, 검찰)에 제출할 것이다.
```

관제실이 고요해졌다.

"이주원이 **증거를 확보하기 위해** 데이터를 빼간 거야?" 도윤이 물었다.

"그렇게 **주장**하고 있어." 진수현이 말했다. "사실 여부는 확인 중이야."

---

김영훈이 입을 열었다.

"서문기 상무에게 보고했다는 건... 확인 가능한가요?"

"이메일이 있다면." 진수현이 말했다.

"이주원의 사내 메일은 우리가 열람 승인을 받았어." 김영훈이 정규를 바라보았다. "정규, 이주원이 서문기 상무에게 보낸 메일을 찾아."

```bash
# 이주원 → 서문기 이메일 검색 (Exchange 로그)
# (실제로는 이메일 서버에서 검색해야 하지만, 여기서는 로그 기반)
```

정규가 이메일 로그를 뒤졌다. 10분 후.

"찾았어." 정규가 말했다.

```
발신: juwon.lee@nexusai.co.kr
수신: moonki.seo@nexusai.co.kr
날짜: 2026-02-20
제목: "[긴급] 학습 데이터 출처 확인 요청"

내용 (요약):
"상무님, training_data_manifest.xlsx의 source_003, source_007, source_012
항목의 출처가 불분명합니다. 일부는 공개 데이터가 아닌 것으로 보이며,
source_012는 AlphaAI의 내부 포맷과 유사합니다.
확인 및 대응 부탁드립니다."
```

"답장은?"

```
발신: moonki.seo@nexusai.co.kr
수신: juwon.lee@nexusai.co.kr
날짜: 2026-02-21
제목: "RE: [긴급] 학습 데이터 출처 확인 요청"

내용:
"이 선임, 수고 많아. 해당 항목은 외부 데이터 파트너를 통해
합법적으로 라이선스한 데이터야. 계약서는 법무팀에 있어.
릴리즈 일정에 집중하자."
```

"서문기 상무가 '합법적 라이선스'라고 대답했어." 김영훈이 말했다.

"하지만 이주원은 **그 말을 믿지 않았다**는 거네." 진호가 말했다.

---

"진수현 실장." 추박하가 말했다. "이주원이 윤리경영실에 제보했다는 건 확인 가능해?"

진수현의 표정이 미묘하게 변했다. "... 확인하겠습니다."

"제보가 실제로 있었고, '문제없음'으로 종결되었다면." 추박하가 말했다. "윤리경영실이 **제대로 조사하지 않은** 건지, 아니면 **위에서 압력이 있었는지**가 중요해."

진수현이 고개를 끄덕였다. 하지만 그녀의 눈은 어딘가를 피하고 있었다.

가람은 그것을 놓치지 않았다.

---

점심 시간. 가람은 구내식당에서 도윤과 밥을 먹고 있었다.

"내부 고발자라..." 도윤이 중얼거렸다. "이주원이 진짜 선한 의도였다면, 우리가 그를 **범인 취급**한 거잖아."

"아직 모르지." 가람이 말했다. "주장과 사실은 달라. 김영훈 과장님이 말했잖아. '확인될 때까지는 전부 가능성'이라고."

"하지만 이메일은 진짜잖아. 서문기 상무에게 보고했다는 게 확인됐잖아."

가람은 젓가락을 내려놓았다.

"도윤 선배. 저도 이상한 게 있어요."

"뭐?"

"서문기 상무가 '합법적 라이선스'라고 했잖아요. 계약서가 법무팀에 있다고. 그럼 **강혜윤 실장이 알고 있어야** 하는데."

"맞아."

"근데 며칠 전 강혜윤 실장이 '학습 데이터의 출처가 합법적인지?'라고 **물었**잖아요. 마치 모르는 것처럼."

도윤의 눈이 커졌다.

"그러면... **계약서가 없거나**, 강혜윤 실장이 **일부러 모른 척**한 거야?"

가람은 수첩에 적었다.

> `서문기: "합법적 라이선스. 계약서는 법무팀에." (2026-02-21)`
> `강혜윤: "학습 데이터 출처가 합법적인지?" (2026-04-09)`
> `강혜윤이 모른다면 → 계약서가 없다? 서문기가 거짓말?`
> `강혜윤이 알면서 물었다면 → 확인 절차? 함정?`

---

오후 3시. 뉴스가 터졌다.

도윤의 교육원 동기 **조은별**(홍보실)에게서 문자가 왔다.

> "도윤아, 너 회사 SOC팀이지? 방금 기자한테서 전화 왔어. 'AI 학습 데이터 논란' 건으로."

도윤이 가람에게 보여주었다.

"기자가 벌써 알고 있어?"

가람이 물었다. "이주원이 기자에게도 연락한 건가?"

"아니면 경찰 쪽에서 새 나왔거나."

---

오후 5시. 서문기 상무가 SOC에 다시 내려왔다. 이번에는 표정이 달랐다. 분노.

"김영훈 과장. 이주원의 진술이 **언론에 유출됐어**. 내일 아침 기사가 나갈 수도 있어."

"저희가 유출한 게 아닙니다."

"알아. 하지만 **언론 대응**이 필요해. 그리고—" 서문기가 김영훈을 바라보았다. "이주원이 말하는 '학습 데이터 문제'는 **사실이 아니야**. 모든 데이터는 합법적으로 수집했어."

추박하가 만화책 뒤에서 말했다.

"상무님. 합법적이라면, **계약서를 보여주시면** 됩니다."

서문기의 눈이 추박하에게 고정되었다.

"... 법무팀에 요청하겠어."

서문기가 나갔다.

추박하가 중얼거렸다. 가람만 들을 수 있는 소리로.

"계약서는 없어."

---

가람은 그날 밤 집에서 수첩을 정리했다.

```
[4월 셋째 주 — 상황 요약]

이주원: "내부 고발자. 학습 데이터에 불법 수집 포함."
서문기: "합법적 라이선스. 계약서는 법무팀에."
강혜윤: 학습 데이터 출처 질문 (알고 있는지 불명)
추팀장: "계약서는 없어." (소곤)
언론: 기사 준비 중 (조은별 → 도윤 → 가람)

[핵심 질문]
이주원은 스파이인가, 내부 고발자인가?
서문기는 진실을 말하고 있는가?
강혜윤은 어느 편인가?
추박하는 언제 genesis를 열 것인가?
```

가람은 수첩을 덮으며 생각했다.

*진실은 하나인데, 모두가 다른 진실을 말하고 있다.*

*하지만 OpsClaw는 하나의 진실만 기록한다.*

*"evidence 없는 완료 주장은 거짓말이다."*

*그렇다면, **evidence가 있는** 주장만이 진실이다.*

*genesis에는 어떤 evidence가 있을까.*
