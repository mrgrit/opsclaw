# Chapter 7: 디지털 유령

---

토요일. 시큐리온 포렌식 팀과 박지후가 이주원의 PC 분석을 계속했다.

sdelete로 보안 삭제된 Documents와 Downloads 폴더는 복구 불가능. 하지만 지후는 **다른 곳**을 파고 있었다.

"사람들이 자주 잊는 게 있어." 지후가 말했다. "**브라우저 캐시**와 **썸네일 캐시**야."

---

### Chrome 프로파일 분석

sdelete는 Documents와 Downloads만 삭제했다. 하지만 Chrome의 프로파일 데이터(`AppData/Local/Google/Chrome/User Data/Default/`)는 **남아 있었다**.

```bash
# Chrome History DB (SQLite)
sqlite3 "/evidence/case-juwon/Chrome/History" \
  "SELECT url, title, last_visit_time FROM urls WHERE url LIKE '%mega%' OR url LIKE '%drive.google%' OR url LIKE '%nexus%' ORDER BY last_visit_time DESC LIMIT 20;"
```

```
https://mega.nz/fm/root | MEGA - My files | 2026-04-09 13:35
https://mega.nz/fm/uploads | MEGA - Uploads | 2026-04-09 13:36
https://drive.google.com/drive/folders/1Ax... | nexus-data-share | 2026-04-09 13:32
https://mail.google.com/mail/u/0/#inbox | Gmail | 2026-04-09 13:28
```

"Google Drive에 'nexus-data-share'라는 폴더에 접근했어." 지후가 말했다. "그리고 바로 Mega에 업로드."

"Google Drive → Mega. **이중 클라우드 경유**로 추적을 어렵게 만든 거야." 시큐리온 전문가가 말했다.

---

### Chrome 다운로드 기록

```bash
sqlite3 "/evidence/case-juwon/Chrome/History" \
  "SELECT target_path, tab_url, end_time FROM downloads ORDER BY end_time DESC LIMIT 10;"
```

```
C:\Users\juwon\Downloads\nexus1_weights_v3.2.tar.gz | https://drive.google.com/... | 2026-04-09 13:33
C:\Users\juwon\Downloads\training_data_manifest.xlsx | https://drive.google.com/... | 2026-04-09 13:33
C:\Users\juwon\Downloads\architecture_doc_v2.pdf | https://drive.google.com/... | 2026-04-09 13:34
```

관제실이 정적에 빠졌다.

`nexus1_weights_v3.2.tar.gz` — **Nexus-1 모델의 가중치 파일**.
`training_data_manifest.xlsx` — **학습 데이터 목록**.
`architecture_doc_v2.pdf` — **아키텍처 문서**.

"이건..." 김영훈이 말을 잇지 못했다.

"**모델 전체를 빼간 거야.**" 추박하가 말했다. 만화책은 아까부터 덮어져 있었다. "가중치, 학습 데이터 매니페스트, 아키텍처 문서. 이 세 가지가 있으면 **Nexus-1을 복제**할 수 있어."

"파일 자체는 sdelete로 삭제됐지만, **다운로드 기록**은 Chrome DB에 남아 있어." 지후가 말했다. "이주원이 Chrome까지 삭제할 **시간이 없었던** 거야."

"30분이라고 했잖아." 가람이 중얼거렸다. "13:30~14:00. 다운로드, 업로드, 삭제, 퇴근. **30분에 모든 걸 하려다 보니 Chrome을 놓친** 거예요."

"맞아." 지후가 고개를 끄덕였다. "범인은 항상 **하나를 놓쳐**."

---

"이걸로 충분해." 김영훈이 말했다. "이주원이 Nexus-1의 핵심 자산 — 모델 가중치, 학습 데이터 매니페스트, 아키텍처 문서 — 을 Google Drive에서 다운로드하여 Mega에 업로드한 증거. Chrome 히스토리와 다운로드 기록이 **디지털 증거**야."

"경찰 신고 가능한 수준인가요?" 진호가 물었다.

"강혜윤 실장에게 확인받자." 김영훈이 전화를 들었다.

---

30분 후. 강혜윤이 관제실에 도착했다. 토요일인데도.

증거를 검토한 후:

"**부정경쟁방지법 위반** (영업비밀 유출) + **산업기술 유출 방지법** 위반. 경찰 신고 가능합니다." 강혜윤이 말했다. "하지만 **한 가지 문제**가 있어요."

"뭔데요?"

"training_data_manifest.xlsx — 학습 데이터 목록이요. 만약 이 학습 데이터 자체에 **문제**가 있다면..."

"무슨 문제요?"

강혜윤이 잠시 침묵했다. 그리고 추박하를 바라보았다.

"학습 데이터의 **출처가 합법적인지**요."

관제실의 공기가 무거워졌다. 가람은 이전에 들었던 말이 떠올랐다.

*"정리된 게 아니라 묻은 겁니다."* — 추박하가 회장에게 한 말.

*"학습 데이터 provenance 문서를 일부러 불완전하게 남긴 정황."* — 서문기 상무에 대한 의혹.

*"퇴직자 노트북에서 AlphaAI 내부 보고서 발견."*

만약 Nexus-1의 학습 데이터에 **불법 수집된 데이터**가 포함되어 있다면?

이주원이 빼간 것은 단순히 "Nexus-1의 기술"이 아니라, **"넥서스AI의 불법 행위의 증거"**일 수도 있다.

그렇다면 이주원은 **산업 스파이인가, 아니면 내부 고발자인가?**

---

강혜윤이 말했다. "신고는 진행하되, **학습 데이터 관련 내용은 경찰에 제출하지 않겠습니다**. 모델 가중치와 아키텍처 문서 유출만으로도 충분해요."

"학습 데이터 목록을 빼는 건..." 김영훈이 말했다.

"**우리 내부에서 먼저 확인**해야 해요. 학습 데이터에 문제가 있다면, 그건 **별도의 문제**예요."

추박하가 조용히 일어서서 관제실을 나갔다. 아무에게도 말하지 않고.

---

가람은 수첩에 적었다.

> `이주원이 빼간 것: 모델 가중치 + 학습 데이터 목록 + 아키텍처 문서`
> `Chrome 다운로드 기록이 유일한 증거 (sdelete가 Chrome을 놓침)`
> `강혜윤: "학습 데이터의 출처가 합법적인지?" ← 핵심 질문`
> `만약 학습 데이터 자체가 불법이라면...`
> `이주원 = 스파이? or 고발자?`
> `추팀장이 아무 말 없이 나갔다. 이건... 처음이다.`
