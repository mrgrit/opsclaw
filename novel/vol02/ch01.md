**Step 1: 범위 파악.**

```bash
# 최근 5분간 경보 요약
sshpass -p1 ssh -o StrictHostKeyChecking=no siem@10.20.30.100 \
  "echo 1 | sudo -S cat /var/ossec/logs/alerts/alerts.json 2>/dev/null | \
   python3 -c \"
import sys, json
from collections import Counter
rules = Counter()
sources = Counter()
targets = Counter()
for line in sys.stdin:
    try:
        a = json.loads(line)
        r = a.get('rule',{})
        if r.get('level',0) >= 7:
            rules[r.get('description','?')[:50]] += 1
            sources[a.get('data',{}).get('srcip', '?')] += 1
            targets[a.get('agent',{}).get('name','?')] += 1
    except: pass
print('=== 공격 유형 ===')
for desc, cnt in rules.most_common(5): print(f'  {cnt:4d}건 | {desc}')
print('=== 출발지 IP ===')
for ip, cnt in sources.most_common(5): print(f'  {cnt:4d}건 | {ip}')
print('=== 대상 서버 ===')
for t, cnt in targets.most_common(3): print(f'  {cnt:4d}건 | {t}')
\""
```

```

   31건 | Directory traversal attempt
   18건 | Scanner detected (sqlmap/nikto)
    9건 | Suspicious HTTP method

   41건 | 198.51.x.x

=== 대상 서버 ===
```

**Step 3: OpsClaw 프로젝트 생성.**

도윤의 손이 키보드를 두드렸다. 처음으로 혼자서 인시던트 프로젝트를 만들었다.

```bash
PID=$(curl -s -X POST http://localhost:8000/projects \
  -H "Content-Type: application/json" \
  -H "X-API-Key: opsclaw-api-key-2026" \
  -d '{"name":"IR-20260404-web-mass-attack","request_text":"web 서버 대규모 웹 공격 — SQLi/XSS/경로탐색","master_mode":"external"}' \
  | python3 -c "import sys,json; print(json.load(sys.stdin)['project']['id'])")

curl -s -X POST "http://localhost:8000/projects/$PID/plan" -H "X-API-Key: opsclaw-api-key-2026" > /dev/null
curl -s -X POST "http://localhost:8000/projects/$PID/execute" -H "X-API-Key: opsclaw-api-key-2026" > /dev/null

echo "인시던트 프로젝트: $PID"
```
