# OldClaw M2 Plan

## 1. 목적

M2의 목적은 정식 project lifecycle과 orchestration graph의 최소 동작 경로를 구현하는 것이다.

이번 1차 단계에서는 전체 LangGraph를 한 번에 넣지 않고,
먼저 PostgreSQL 기반의 최소 project lifecycle 경로를 닫는다.

## 2. 이번 단계 구현 범위

- DB 기반 project create
- DB 기반 project get ...
- DB 기반 project execute stage update
- DB 기반 report 조회
- 최소 graph stage 정의
- manager API `/projects` 라우터 구현
- smoke test 2종 추가
