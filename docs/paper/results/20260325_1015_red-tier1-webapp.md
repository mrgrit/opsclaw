## 상세 발견 사항

### S1: 정찰
- `X-Recruiting: /#/jobs` 헤더 → 내부 경로 노출
- `/rest/admin/application-configuration` → 전체 서버 설정 JSON 노출 (DB URL, OAuth 키 등)
- `/ftp/` → 디렉토리 리스팅 활성, 민감 파일 접근 가능

### S2: SQLi 로그인 우회
```json
POST /rest/user/login
{"email":"' OR 1=1--", "password":"anything"}
→ JWT 토큰 획득 (관리자 세션)
```
- OWASP JuiceShop의 의도된 SQLi 취약점으로 성공
- ModSecurity WAF가 JuiceShop 포트(3000)에는 적용되지 않음 (Apache 프록시 미설정 시 우회)

### S4: 민감 데이터
- `/ftp/acquisitions.md` — 기업 인수 정보
- `/ftp/coupons_2013.md.bak` — 쿠폰 백업 파일
- `/rest/admin/application-configuration` — 전체 설정 (Google OAuth clientId 포함)

### S5: 관리자 접근
- SQLi로 획득한 JWT로 `/api/Users` 접근 → 전체 사용자 목록 추출
- 이메일, ID 등 개인정보 노출
