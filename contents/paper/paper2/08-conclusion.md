# 8. 결론 (Conclusion)

본 논문은 LLM 에이전트(Claude Code)를 활용한 자율 모의해킹 및 방어 자동화의 가능성과 한계를 MITRE ATT&CK 4-Tier 실험을 통해 실증적으로 평가하였다. 실환경 보안 인프라 4대 서버(IPS, WAF, SIEM, 웹 애플리케이션)에서 수행한 실험의 주요 결과는 다음과 같다.

**Red Team.** LLM 에이전트는 21개 ATT&CK 기법 중 17개(81%)를 성공적으로 실행하여 77.8%(21/27)의 공격 성공률을 달성하였다. 웹 애플리케이션 공격 체인(91.7%), 네트워크 우회(75.0%), 권한 상승(75.0%), SIEM 탐지 우회(71.4%)의 4-Tier에서 모두 유의미한 성과를 보였다. 특히 sudo NOPASSWD ALL 취약점 발견, 다중 지속성 메커니즘 설치, SIEM 무력화 체인 구성 등 복잡한 다단계 공격을 자율적으로 수행하였다.

**Blue Team.** LLM 에이전트는 7개의 Wazuh 커스텀 탐지 룰과 1개의 Suricata IPS 시그니처를 자율적으로 생성하였으며, 모든 룰이 logtest 검증을 통과하였다(방어 점수 12/16). ATT&CK 기법 매핑의 정확성과 룰 품질이 확인되었으나, 실시간 탐지 검증은 인프라 이슈로 미완이다.

**Purple Team.** 4회전의 공격-방어 반복 사이클을 통해 무방비 상태에서 전면 방어까지 점진적으로 강화하였다. 총 8건의 취약점(Critical 3, High 3, Medium 2)을 발견하고, T4에서 모든 공격 경로를 차단하는 데 성공하였다. LLM 에이전트는 이전 방어를 우회하는 새로운 공격 벡터를 적응적으로 탐색하는 행동을 보여, 규칙 기반 에뮬레이션 도구와의 차별점을 실증하였다.

본 연구는 LLM 에이전트가 모의해킹의 Red/Blue/Purple 전 단계에서 유의미한 자동화를 제공할 수 있으나, 도구 가용성 의존, 명령 이스케이핑 오류, 컨텍스트 유지 한계 등 현실적 제약이 존재함을 보여준다. 향후 다중 LLM 비교, CALDERA 연동, 실시간 탐지 루프 구현을 통해 자율 모의해킹의 실용성을 더욱 확대할 계획이다.
