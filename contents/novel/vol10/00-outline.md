# OpsClaw 소설 10권 — "새벽" 아웃라인

## 부제: The Dawn After

## 배경
genesis 데이터가 검찰에 제출되었다. 추박하는 증인이자 공범으로 조사를 받고 있다.
서문기는 체포되었고, 회장이 연루되었다. 넥서스AI는 대규모 소송에 직면한다.
SOC팀은 모든 것이 무너지는 가운데에서도 회사를 유지해야 한다.

## 주요 인물 상태 (10권 시작 시점)

| 인물 | 상태 |
|------|------|
| 한가람 | SOC 분석가 1년차 → 핵심 증인 겸 분석가 |
| 김영훈 | SOC팀장 대리, 팀을 지키며 정상 운영 유지 |
| 강혜윤 | 포렌식 분석관, 최종 보고서 작성 주도 |
| 이주원 | CISO 복직 후 법정 증인, 개인정보보호 전문가 |
| 추박하 | 증인 겸 공범, 조사 중 |
| 서문기 | 체포, 구속 |
| 조은별 | SOC 분석가, 위기 관리 최전선 |
| 정규 | SOC 주니어, Wazuh/탐지 룰 담당 성장 |
| 회장 | 연루 의혹, 수사 대상 |

## 전체 구조: 3부

- Ch 3: 언론 폭풍 — 전면 언론 보도, 위기 관리
- Ch 4: 주주총회 — 회사 생존 위기

### 2부: 재건 (Ch 5-8) — 폐허 위에 다시 짓다
- Ch 5: 모니터링 — SOC팀의 묵묵한 일상
- Ch 6: 코드로 다시 짓다 — 인프라 재설계
- Ch 7: Wazuh가 지킨 것 — SIEM 로그의 법정 활용
- Ch 8: 규칙을 만드는 사람 — 탐지 룰 재설계

### 3부: 새벽 (Ch 9-12) — 1년 후, 그리고 시작
- Ch 9: 에필로그: 1년 후 — 팀

- Ch 12: 새벽 — 최종장, 1권 1장과 대칭 구조

## 장별 상세

### Ch 1: 증거 제출
- **강의 연계**: C3W14(심화) 보고서 작성
- **핵심 장면**: 강혜윤이 검찰청에서 evidence_pack을 전달. 보고서 작성의 기술적 엄밀함.
- **감정선**: 긴장, 책임감, "기록이 진실을 증명한다"

### Ch 2: 법정의 아침
- **강의 연계**: C4W09(심화) GDPR/개인정보
- **핵심 장면**: 이주원이 증인석에 선다. 개인정보보호법 위반의 구체적 내용.
- **감정선**: 두려움과 용기, 전문성이 진실을 뒷받침

### Ch 3: 언론 폭풍
- **강의 연계**: C3W08(심화) 점검 보고서
- **핵심 장면**: "넥서스AI 20년의 거짓말" 보도. 조은별의 위기 관리.
- **감정선**: 혼란, 분노, 그리고 묵묵한 대응

### Ch 4: 주주총회
- **강의 연계**: C4W13(심화) SOC 2/HIPAA
- **핵심 장면**: 긴급 주주총회. 컴플라이언스 질의. 회사 존폐 위기.
- **감정선**: 압박, 생존, "규정을 지킨 것이 회사를 살린다"

### Ch 5: 모니터링
- **강의 연계**: C6W13(심화) 클라우드 모니터링
- **핵심 장면**: 사건 수습 중에도 인프라는 멈추지 않는다. SOC팀의 일상.
- **감정선**: 고요한 긍지, 묵묵함

### Ch 6: 코드로 다시 짓다
- **강의 연계**: C6W14(심화) IaC 보안
- **핵심 장면**: 넥서스AI 인프라 재설계. Terraform + 투명한 파이프라인.
- **감정선**: 희망, "이번에는 제대로"

### Ch 7: Wazuh가 지킨 것
- **강의 연계**: C2W09 Wazuh(심화)
- **핵심 장면**: Wazuh 로그가 법정에서 결정적 증거로. SIEM의 가치 증명.
- **감정선**: 자부심, 기술에 대한 신뢰

### Ch 8: 규칙을 만드는 사람
- **강의 연계**: C2W10(심화) Wazuh 룰
- **핵심 장면**: 새 보안 정책 하의 탐지 룰 재설계. 정규의 성장.
- **감정선**: 성장, 책임감, "기본부터 다시"

### Ch 9: 에필로그: 1년 후 — 팀
- **강의 연계**: C5W01(심화) SOC 개론
- **핵심 장면**: 김영훈 팀장 승진, 추박하 퇴직, 새 신입.
- **감정선**: 따뜻함, 변화, 지속

### Ch 10: 에필로그: 1년 후 — 사람들
- **강의 연계**: C8W12(최종) AI 윤리
- **핵심 장면**: 각 인물의 1년 후. 추박하 집행유예, 서문기 실형, 이주원 복직, 회장 재판 중.
- **감정선**: 쓸쓸함, 수용, 각자의 길

### Ch 11: 에필로그: 1년 후 — OpsClaw
- **강의 연계**: C7W09(최종) OpsClaw
- **핵심 장면**: OpsClaw v2 오픈소스 발표. "기록은 진실이다" 공개 슬로건.
- **감정선**: 자부심, 공유, 새로운 시작

### Ch 12: 새벽
- **강의 연계**: C1W01(최종) 보안 개론
- **핵심 장면**: 가람의 2년차 첫 출근. 새 신입에게 SOC 안내. 추박하의 빈 자리에 만화책. "기록은 진실이다."
- **감정선**: 따뜻한 쓸쓸함, 순환, 완결
- **구조**: 1권 1장과 완벽한 대칭 — 가람이 신입이었던 것처럼, 이제 선배로서 새 사람을 맞이한다.
- **마지막 문장**: "기록은 진실이다." — 전 10권의 마침표
