### 오전: 모니터링 체계 최종 점검

이사회까지 5시간. 김영훈은 genesis의 모니터링 체계를 최종 점검했다.

"이사회 결과와 관계없이, genesis는 **지속적으로 모니터링**되어야 해." 김영훈이 말했다. "클라우드 모니터링 개념을 정리하면서 마무리하자."

```
[클라우드 모니터링 — 핵심 서비스]

1. CloudWatch (AWS)
   - 메트릭 수집: CPU, 메모리, 네트워크, 디스크
   - 경보(Alarm): 임계값 초과 시 SNS 알림
   - 대시보드: 실시간 시각화
   - 로그 그룹: 애플리케이션 로그 중앙 수집
   → genesis S3 버킷의 접근 패턴 모니터링에 사용

2. CloudTrail (AWS)
   - API 호출 감사 로그
   - 누가(Who), 언제(When), 무엇을(What) 했는지 기록
   - S3에 로그 저장 → 변조 방지
   - 이상 탐지: 평소와 다른 API 호출 패턴 감지
   → genesis 관련 모든 AWS API 호출을 추적

3. GuardDuty (AWS)
   - 위협 탐지 서비스
   - CloudTrail + VPC Flow Logs + DNS Logs를 분석
   - 이상 행위 자동 탐지 (예: 새벽 3시 미국 IP에서 접근)
   → genesis 버킷에 대한 이상 접근 자동 경보

4. Config (AWS)
   - 리소스 설정 변경 추적
   - "이 S3 버킷의 설정이 언제 바뀌었나?"
   - 컴플라이언스 규칙: "퍼블릭 버킷이 없어야 한다"
   → genesis 버킷의 설정 변경을 자동 감지
```

김영훈이 CloudWatch 경보를 설정했다. 오브젝트 수 이상 증가 시 SNS 알림. 그리고 CloudTrail에서 genesis 버킷 관련 이벤트를 필터링했다.

5월 25일부터 오늘까지의 모든 접근 — genesis-bkup IAM 사용자에서만. PutObject 5건. 외부 사용자 접근 **0건.**

"깨끗해." 김영훈이 말했다.

"**감시자가 제대로 작동하고 있어.**" 추박하가 말했다. 처음으로 안도의 표정이 스쳤다.

### 오후 2시: 이사회

관제실에서 이사회를 직접 볼 수는 없었다. 추박하는 이사회 회의실 앞 대기석에 앉아 있었고, 관제실에는 김영훈과 팀원들이 남았다.

15:30. 결론 (예상).

가람이 시계를 봤다. 14:07.

"코드를 짜는 게 낫다." 정규가 말했다. 실제로 코드를 짜고 있었다. K8s NetworkPolicy YAML.

김영훈은 아무것도 하지 않고 앉아 있었다. 화면을 보는 것 같았지만, 커서가 움직이지 않았다.


### 16:00: 결과

강혜윤과 추박하가 관제실에 들어왔다.

강혜윤은 평소와 같은 표정이었다 — 차분하고, 정리된. 하지만 눈가에 미세한 안도가 있었다. 추박하는 넥타이를 이미 풀어 손에 들고 있었다.

"결과를 말씀드리겠습니다." 강혜윤이 말했다.

관제실이 숨을 죽였다.

```
[이사회 결정 사항 — 2026.06.04]

1. 자진 신고 (Option A 채택)
   - Nexus-1 학습 데이터 문제를 개인정보보호위원회에 자진 신고
   - 경쟁사 데이터 불법 수집 건은 별도 법률 검토 후 신고 여부 결정

2. 외부 감사
   - SecureEdge에 genesis 독립 검증 의뢰 (한승우 대표)
   - 법무법인에 법적 리스크 평가 의뢰

3. Nexus-1 릴리즈 연기
   - 학습 데이터 세탁 완료 시까지 릴리즈 보류
   - 클린 데이터셋으로 재학습 검토

4. 인사 조치
   - 서문기 상무: 사표 수리 보류, 내부 조사 완료 시까지 직위 해제
   - 추박하 팀장: genesis 관리 및 증거 보전 책임자로 지정

5. genesis 보존
   - genesis의 PoW 체인을 법적 증거로 보전
   - 현재 Docker 환경 유지 (환경 변경 금지)
   - 외부 백업(S3) 보안 강화 후 유지
```

"**Option A.**" 김영훈이 말했다. "자진 신고."

"이사회가 만장일치로 결정했습니다." 강혜윤이 말했다. "자진 신고가 **벌금 감면**과 **브랜드 회복**에 유리하다는 법무팀의 의견이 결정적이었습니다."

"서문기 상무의 사표는?" 정규가 물었다.

"**수리 보류.** 내부 조사가 끝날 때까지 회사를 떠날 수 없습니다." 강혜윤이 말했다. "도망가려 했지만, 이사회가 막았습니다."

도윤이 작게 주먹을 쥐었다. "**쥐를 배 위에서 잡았네요.**"

### 추박하

보고가 끝나고, 팀원들이 웅성거리는 가운데.

추박하가 창가에 서 있었다. 정장 재킷은 의자에 걸려 있었고, 셔츠 소매를 걷어올린 팔이 석양 빛을 받고 있었다.

강혜윤이 다가갔다.

"추 팀장님."

"응."

"이사회가 Option A를 선택했습니다."

"들었어."

"20년이에요." 강혜윤이 말했다. 목소리가 평소보다 조금 부드러웠다. "20년간 기록한 것이 오늘 빛을 봤습니다."

추박하가 잠시 창밖을 바라보았다. 넥서스AI 사옥 앞의 가로수가 초여름 바람에 흔들리고 있었다.

"고마워, 혜윤아."

강혜윤이 살짝 놀랐다. 추박하가 이름을 부르는 건 드문 일이었다.

"아직 끝난 게 아니에요." 강혜윤이 말했다. "자진 신고, 외부 감사, 재학습 — 갈 길이 멀어요."

"알아." 추박하가 말했다. "하지만 오늘은... 여기까지만 걱정하자."

추박하가 돌아서며 관제실을 바라보았다. 민재가 커피를 나눠주고, 도윤이 뭔가 농담을 하고, 정규가 무표정하게 웃고, 진호가 지후와 토론하고, 가람이 수첩에 뭔가를 적고 있었다.

"좋은 팀이야." 추박하가 중얼거렸다. 강혜윤만 들을 수 있는 목소리로.

### 저녁: 뒤풀이

추박하가 또 치킨을 시켰다.

"이번에도 추팀장님이 쏘시는 건가요?" 민재가 물었다.

"이사회 결과 축하니까." 추박하가 말했다.

"축하라기보다는 **시작** 아닌가요?" 김영훈이 말했다.

"시작이니까 더 먹어야지." 추박하가 말했다.

치킨을 먹으면서 가람이 물었다.

"추팀장님. genesis가 이제 **공식적으로 존재하는 시스템**이 된 거예요?"

"이사회에 보고했으니까." 추박하가 말했다. "**20년간 숨겨져 있던 감시자**가 세상에 나온 거야."

"감시자." 가람이 되뇌었다.

"genesis는 원래 넥서스AI를 **보호하기 위해** 만들었어." 추박하가 치킨 뼈를 내려놓으며 말했다. "하지만 결국 넥서스AI의 **잘못을 증명하는** 도구가 됐지. 아이러니야."

"아이러니가 아니에요." 가람이 말했다. "**정직한 기록은 어느 방향으로든** 진실을 가리키니까요."

추박하가 가람을 바라보았다. 그리고 고개를 끄덕였다.

"맞아. 그게 genesis의 본질이야. **감시자는 편을 들지 않는다.** 기록할 뿐이지."

치킨 냄새와 웃음소리. 평범한 저녁. 하지만 오늘 이 관제실에서 **20년의 비밀이 세상에 나오기 시작했다.**

아직 끝나지 않았다. 자진 신고는 시작일 뿐이다. 법적 절차, 외부 감사, 세상의 반응 — 모든 것이 앞에 놓여 있다.

하지만 오늘만큼은.

추박하가 치킨을 뜯으며 웃고, 민재가 콜라를 쏟아 도윤에게 소리를 듣고, 정규가 냅킨에 K8s 아키텍처를 그리고, 김영훈이 강혜윤과 조용히 다음 단계를 논의하는 — 이 평범한 저녁이, 가람에게는 충분했다.

genesis의 이야기는 계속된다.
