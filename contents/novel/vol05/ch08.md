"**그래서** 하는 거야." 김영훈이 말했다. "팀원들이 지쳐 있어. Nexus-1 테스트, ISO 점검, SecureEdge 대응. 일주일 내내 긴장만 했잖아. 한 번쯤 **재미있게** 보안을 해야지."

### CTF 규칙

김영훈이 규칙을 설명했다.

```
[넥서스AI SOC 내부 CTF — "Operation CLAW"]

규칙:
  - 팀: 2~3인 1조 (총 3팀)
  - 시간: 오전 10시 ~ 오후 4시 (6시간)
  - 환경: 내부 테스트 서버 (실 서버 아님)
  - 카테고리: Web, Forensics, Crypto, Network, Misc

팀 구성:
  Team Alpha: 정규 + 가람 (분석 + 기록)
  Team Beta:  민재 + 도윤 (자동화 + 관제)
  Team Gamma: 진호 + 지후 (OSINT + 포렌식)

점수: 문제당 100~500점, 시간 보너스 있음
상품: 추팀장이 **점심 쏘기** (1등 팀)
```

"상품이 점심이에요?" 도윤이 웃었다.

"추팀장님이 밥을 사는 건 **역사적 사건**이야." 민재가 말했다. "내가 5년 동안 한 번도 못 봤어."

"그만큼 **이기고 싶겠지.**" 추박하가 웃었다.

### 문제 풀이

김영훈이 JuiceShop과 별도 CTF 서버에 문제를 준비해두었다. 10문제.

**Web 1 (100점): "첫 번째 주스"**

```
JuiceShop(http://10.20.30.80:3000)에서 관리자 계정에 로그인하라.
FLAG 형식: FLAG{admin_email}
```

가람이 먼저 시도했다.

"1권에서 배운 SQL Injection으로 해볼게요." 가람이 JuiceShop 로그인 페이지에 입력했다.

```
Email: ' OR 1=1--
Password: (아무거나)
```

"**로그인 성공!** 관리자 계정은 admin@juice-sh.op이에요."

"FLAG{admin@juice-sh.op}" 정규가 제출했다. **100점.**

"1권 때 배운 거 아직 기억하네." 정규가 고개를 끄덕였다.

---

**Forensics 1 (200점): "사라진 파일"**

```
주어진 메모리 덤프에서 삭제된 파일의 이름을 찾아라.
파일: /ctf/challenges/mem_dump.raw
FLAG 형식: FLAG{filename.ext}
```

지후가 포렌식 도구를 열었다.

```bash
# Volatility로 메모리 분석
volatility -f /ctf/challenges/mem_dump.raw --profile=LinuxUbuntu2204x64 linux_find_file -F "/tmp"
```

"삭제된 임시 파일이 있어요. **/tmp/secret_plan.docx**" 지후가 말했다.

"FLAG{secret_plan.docx}" 진호가 제출. **200점.**

---

**Network 1 (300점): "유령의 속삭임"**

```
주어진 pcap 파일에서 DNS 터널링으로 유출된 데이터를 추출하라.
파일: /ctf/challenges/ghost.pcap
FLAG 형식: FLAG{extracted_data}
```

정규가 tcpdump로 분석했다.

```bash
tcpdump -r /ctf/challenges/ghost.pcap -n 'udp port 53' | head -20
```

"DNS 쿼리 도메인에 데이터가 인코딩되어 있어요."

```
646174615f6578.evil.com  → hex 디코딩 → "data_ex"
66696c7472617465.evil.com → "filtrate"
645f7365637265.evil.com → "d_secre"
74.evil.com → "t"
```

"조합하면: **data_exfiltrated_secret**" 가람이 말했다.

"FLAG{data_exfiltrated_secret}" 정규가 제출. **300점.**

"3권에서 배운 DNS 터널링이었네요." 가람이 말했다. "C2 비콘이 DNS로 데이터를 빼돌리던 것과 같은 원리."

---

**Crypto 1 (400점): "깨진 자물쇠"**

```
아래 해시에서 원본 비밀번호를 찾아라.
Hash: 5f4dcc3b5aa765d61d8327deb882cf99
Algorithm: MD5
FLAG 형식: FLAG{password}
```

민재가 즉시 대답했다. "이건 MD5('password')야."

"어떻게 바로 아세요?" 도윤이 놀랐다.

"이건 **세계에서 가장 유명한 MD5 해시**야. 보안쪽에서 일하면 외워져."

"FLAG{password}" 민재가 제출. **400점.**

"1권에서 SSH 비밀번호가 '1'이었던 것보다는 낫네요." 가람이 중얼거렸다.

"**세상에서 가장 흔한 비밀번호 Top 5**: 123456, password, 12345678, qwerty, 123456789." 정규가 말했다. "이 중 하나라도 쓰고 있는 사람이 있으면 지금 바꿔."

---

**Misc 1 (500점): "OpsClaw의 비밀"**

```
OpsClaw의 PoW 체인에서 특정 블록의 nonce를 찾아라.
조건: block_hash가 "0000"으로 시작하는 블록 중,
      agent_id가 "http://10.20.30.1:8002"이고
      created_at이 2026-04-18인 블록의 nonce.
FLAG 형식: FLAG{nonce_value}
```

"**OpsClaw 문제!**" 민재가 눈을 빛냈다. "이건 내 영역이야."

```bash
curl -s -H "X-API-Key: opsclaw-api-key-2026" \
  "http://localhost:8000/pow/blocks?agent_id=http://10.20.30.1:8002" \
  | python3 -c "
import sys, json
blocks = json.load(sys.stdin).get('blocks', [])
for b in blocks:
    if b['block_hash'].startswith('0000') and '2026-04-18' in b.get('created_at',''):
        print(f'nonce={b[\"nonce\"]} hash={b[\"block_hash\"][:20]}...')
        break
"
```

```
nonce=48723 hash=0000a7f3b2e1c9d8f6...
```

"FLAG{48723}" 민재가 제출. **500점.**

"nonce 48723이면, **48,723번** 해시를 계산해서 '0000'으로 시작하는 값을 찾은 거예요." 민재가 가람에게 설명했다. "이게 PoW의 본질이야. 쉽게 만들 수 없지만, **검증은 한 번**이면 돼."

"**가람이 1등이야!**" 도윤이 환호했다.

"정규 선배 덕분이에요." 가람이 겸손해했다.

"가람이가 DNS 터널링 문제를 풀었어." 정규가 말했다. "3개월 전의 가람이었으면 못 풀었을 거야."

"약속대로 **점심은 내가 쏜다.** Alpha 팀 원하는 곳."

"민재야. 3등은 조용히 해."

관제실에 웃음이 터졌다. 오랜만에.

저녁. CTF 뒤풀이로 소고기를 먹으러 갔다. 추박하, 김영훈, 정규, 가람, 민재, 도윤, 진호, 지후. 관제실 전원.

고기를 굽는 동안 자연스럽게 대화가 흘렀다.

"과장님, CTF 문제 만드시느라 언제 주무셨어요?" 도윤이 물었다.

"안 잤어." 김영훈이 담담하게 말했다.

"... 역시."

"근데 OpsClaw 문제 500점짜리는 누가 만들었어요?" 민재가 물었다.

김영훈이 추박하를 바라보았다. 추박하가 고기를 집으며 시치미를 뗐다.

"PoW nonce를 찾으라는 문제 아이디어는 **팀장님**이 내셨거든." 김영훈이 말했다.

"팀장님이요?" 민재가 놀랐다. "팀장님이 CTF 문제를 만드신 거예요?"

"아이디어만." 추박하가 말했다. "만든 건 영훈이가."

"팀장님도 CTF 좋아하세요?" 가람이 물었다.

추박하가 잠시 생각했다.

"CTF는 **게임이야.** 보안을 게임처럼 할 수 있다면, 재미있게 배울 수 있지." 추박하가 말했다. "현실의 보안은 게임이 아니지만, **게임에서 배운 기술은 현실에서도 쓸 수 있어.**"

"그게 **시뮬레이션 학습**이죠." 가람이 말했다. "비행기 조종사도 시뮬레이터로 먼저 훈련하잖아요."

"맞아. CTF는 **보안의 비행 시뮬레이터**야." 추박하가 고개를 끄덕였다.

고기가 다 익었다. 추박하가 가람의 접시에 고기를 올려주었다.

"오늘 잘했어." 추박하가 말했다.

가람은 "감사합니다"라고 말하려다, 추박하의 눈에서 무언가를 읽었다. 피로도, 만족도 아닌 — **안도.** 팀이 웃고 있다는 것에 대한 안도.

*추팀장은 genesis의 무게를 지고 있으면서, 팀원들이 웃을 수 있게 해주고 있다.*

*이건 게으름이 아니다. 이건 **배려**다.*
