# Chapter 4: 유리창 너머

---

일요일. 송진호는 자택에서 노트북을 열었다.

어제의 공격을 계속 파고 있었다. 5개 IP의 연관 관계. Passive DNS. WHOIS. 다크웹.

진호의 텔레그램 채널에서 흥미로운 메시지를 발견했다.

```
[채널: threat_intel_kr]
@shadow_analyst: AI 모델 API 엔드포인트 리스트 판매. GPT-4, Claude, Nexus-1 포함.
가격: 0.5 BTC. DM.
2026-04-02
```

**Nexus-1.**

2일 전. 공격 이틀 전에 누군가가 Nexus-1의 API 엔드포인트를 **다크웹에서 판매**하고 있었다.

진호의 손이 떨렸다.

---

월요일 오전. 진호가 관제실에 뛰어들어왔다.

"과장님. 다크웹에서 Nexus-1 API 엔드포인트가 **판매**되고 있었습니다."

김영훈이 화면을 확인했다. 텔레그램 스크린샷.

"shadow_analyst... 이 계정 프로파일은?"

"계정 생성 2025년 11월. 이전 포스팅들을 보면 기업 API 엔드포인트, VPN 크리덴셜, 내부 네트워크 맵을 거래하는 **IAB(Initial Access Broker)**야."

"IAB." 김영훈이 되뇌었다. "초기 접근권을 판매하는 브로커. 직접 공격하지 않고, **침투 경로를 다른 공격자에게 판다**는 거지."

진호가 MITRE ATT&CK Navigator를 열었다.

```
[업데이트된 TTP 매핑]
TA0042 Resource Development:
  T1583.001 Domains — nexus-cdn.com 유사 도메인
  T1583.003 Virtual Private Server — Linode, OVH VPS

TA0001 Initial Access:
  T1190 Exploit Public-Facing Application — Nexus-1 API 노출 악용
  T1110.001 Password Guessing — SSH 브루트포스

TA0009 Collection:
  T1119 Automated Collection — 17건 completions 자동 호출

TA0010 Exfiltration:
  T1041 Exfiltration Over C2 Channel — API 응답으로 데이터 유출

신규:
TA0042 Resource Development:
  T1650 Acquire Access — IAB를 통해 API 접근 정보 구매/판매
```

"이건 **최소 2단계 구조**야." 진호가 말했다. "1단계: IAB가 넥서스AI 내부 정보를 수집하여 판매. 2단계: 구매자가 그 정보로 실제 공격."

"IAB에게 정보를 제공한 건?" 김영훈이 물었다.

"**내부자**." 진호가 말했다. "서버 사용자명 'web', Nexus-1 API 경로 `/v1/nexus/`... 이건 외부에서 알 수 있는 정보가 아니야."

---

추박하가 만화책을 내려놓았다.

"진호야."

"네, 팀장님."

"그 텔레그램 채널, 언제부터 모니터링하고 있었어?"

진호가 잠시 멈추었다. "... 6개월 전부터요."

"6개월이면 회사에서 시킨 건 아니지."

"개인적으로 하고 있었습니다."

추박하가 고개를 끄덕였다. "좋은 습관이야. 계속해."

그리고 다시 만화책을 집었다. 하지만 가람은 보았다 — 추박하가 만화책 사이에 끼운 작은 쪽지에 뭔가를 적는 것을.

---

오후. 김영훈이 인시던트 보고서를 업데이트했다.

```bash
curl -s -X POST "http://localhost:8000/projects/$PID/completion-report" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: opsclaw-api-key-2026" \
  -d '{
    "summary": "Nexus-1 API 침해 — IAB 경유 다단계 공격 확인",
    "outcome": "partial",
    "work_details": [
      "5개 IP 차단",
      "Nexus-1 프록시 비활성화",
      "다크웹 IAB 판매 확인",
      "ATT&CK TTP 매핑 완료"
    ],
    "issues": [
      "Nexus-1 응답 로그 부재 — 유출 범위 확인 불가",
      "IAB에 정보를 제공한 내부자 미식별",
      "서문기 상무의 외부 신고 거부"
    ],
    "next_steps": [
      "내부자 위협 조사 확대",
      "Nexus-1 API 인증 강화",
      "전 서버 접근 로그 상관 분석"
    ]
  }'
```

"completion-report 제출 완료." 민재가 말했다. "OpsClaw에 영구 기록됐어."

김영훈이 고개를 끄덕였다. "이 보고서는 **삭제할 수 없어**. 누가 '그런 사고는 없었다'고 해도, OpsClaw에 증거가 남아."

가람은 그 말에 소름이 돋았다. 추팀장이 말했던 것이 떠올랐다.

*"검증 가능한 영수증."*
*"절대 삭제하면 안 돼."*

OpsClaw는 보안 도구인 동시에, **회사의 양심**이었다. 기록은 거짓말을 하지 않는다.
