# Chapter 12: 불꽃

---

금요일 밤. 넥서스AI 본사가 텅 빈 시간.

김영훈은 관제실에 혼자 남아 있었다. 모든 팀원이 퇴근한 후.

그는 이주원의 이메일 로그를 분석하고 있었다. 진수현 실장의 허가를 받은 범위 내에서.

```
[이주원 이메일 분석 — 비공개]

수신/발신 패턴 (최근 3개월):
- 사내 이메일: 일 평균 25건 (정상)
- 외부 이메일: 일 평균 3건 (정상 범위)
- 특이점: 외부 수신 중 Gmail 발신자 1명이 반복 (jw.research.2025@gmail.com)

jw.research.2025@gmail.com과의 교신:
- 01/15: "논문 공동 작업 관련" (첨부: 없음)
- 02/03: "데이터셋 공유 링크" (첨부: Google Drive URL)
- 02/28: "미팅 일정" (첨부: 없음)
- 03/15: "최신 벤치마크 결과" (첨부: Mega 링크)
```

김영훈의 눈이 두 번째와 네 번째 메일에 고정되었다.

"Google Drive 링크... Mega 링크..."

퇴직자 노트북에서 발견된 파일의 다운로드 소스와 **같은 패턴**이었다.

```
퇴직자 노트북:
  https://drive.google.com/file/d/1Ax3B...  (AlphaAI 보고서)
  https://mega.nz/file/XXXXXX              (data_package)

이주원 이메일:
  02/03: Google Drive URL
  03/15: Mega 링크
```

"같은 유형의 클라우드 저장소. 같은 패턴의 공유." 김영훈이 중얼거렸다.

"jw.research.2025@gmail.com... 'jw'는 이니셜인가? 이주원(Lee Juwon)이면 'lj'일 텐데, 'jw'는..."

김영훈은 OSINT 기법으로 이메일 주소를 추적했다.

```bash
# Gmail 프로필 확인 (Google People API)
# jw.research.2025@gmail.com → 프로필 이름: "JW Kim"
```

"JW Kim."

김영훈은 퇴직자 노트북의 파일을 다시 확인했다.

```
competitor_report_2025.pdf
  Creator: Kim, Jihye (AlphaAI Research)
```

**Kim, Jihye.** AlphaAI 연구소. 이니셜 JH.

*JW Kim ≠ Kim Jihye?*

하지만 "JW"가 "Jihye"의 변형은 아니다. 별개의 인물.

*그렇다면 jw.research.2025는 누구인가?*

---

김영훈은 더 파고 싶었다. 하지만 여기서 멈추어야 했다. 이메일 내용 자체를 열어보려면 **법원 영장**이나 **회사 경영진의 공식 승인**이 필요했다. 현재 갖고 있는 것은 "이메일 로그(발신/수신 메타데이터)"에 대한 열람 승인뿐이었다.

"증거는 **쌓이고 있지만, 결정적 한 발이 없어**." 김영훈이 한숨을 쉬었다.

---

관제실 문이 열렸다. 추박하가 들어왔다.

"영훈아. 아직 있었어?"

"네. 이주원 이메일 분석 중이었습니다."

추박하가 옆자리에 앉았다. 처음으로 만화책 없이.

"뭐가 나왔어?"

김영훈이 분석 결과를 보여주었다. Google Drive, Mega 링크, jw.research.2025.

추박하가 화면을 읽었다. 표정이 변하지 않았다.

"영훈아."

"네."

"이 조사는 **길어질 거야**. 이주원만의 문제가 아닐 수도 있어."

"알고 있습니다."

"그리고 이 과정에서 네가 **불편한 진실**을 발견할 수도 있어."

김영훈이 추박하를 바라보았다.

"불편한 진실이요?"

"이 회사에는... 네가 모르는 역사가 있어." 추박하가 조심스럽게 말했다. "내가 20년간 여기 있으면서 본 것들이 있어."

"팀장님이 genesis에 기록하는 게 그건가요?"

추박하의 눈이 커졌다. 아주 잠깐. 그리고 돌아왔다 — 평소의 나른한 표정으로.

"누가 말해줬어?"

"아무도요. 제가 봤습니다."

긴 침묵.

추박하가 미소를 지었다. 이번에는 진짜 웃음이었다.

"넌 관찰력이 좋아. 가람이도."

"가람이가요?"

"가람이는 수첩에 적고 있고, 넌 머릿속에 적고 있지."

김영훈이 입을 다물었다.

추박하가 일어서며 말했다.

"영훈아. 네가 진실을 찾으면, **네가 판단해**. 나는 기록만 할 거야. 판단은 네 몫이야."

그는 관제실을 나갔다.

---

김영훈은 빈 관제실에 혼자 앉아 있었다.

모니터의 녹색 숫자가 깜빡였다.

```
OpsClaw PoW Chain — Blocks: 47,879
Valid: true
Tampered: 0
```

47,879개의 블록. 20년의 기록.

김영훈은 처음으로 — genesis 프로젝트에 **접근해보고 싶다**는 생각을 했다.

하지만 접근 권한이 없었다. 그리고 추박하가 "기록만 한다"고 했다면, 그 기록을 열어보는 것은 **추박하의 허락 없이는 안 되는 일**이었다.

*아직은.*

김영훈은 터미널을 닫고 불을 껐다.
