# Chapter 12: 열쇠

---

금요일 저녁 7시. 넥서스AI SOC 관제실.

이번 주가 끝나려 하고 있었다. 관제실에서는 주말 교대 일정을 정하고 있었다.

"이번 주말 누가 당번이야?" 도윤이 물었다.

"나." 정규가 말했다.

"정규 선배 지난주도 했잖아요." 가람이 말했다.

"상관없어. 집에 있으나 여기 있으나." 정규가 모니터를 바라보며 말했다.

"그건 좀 슬픈 얘긴데요." 민재가 커피를 들고 다가왔다. "정규야, 주말에 좀 쉬어. 내가 대신할게."

"네가?" 정규가 민재를 바라보았다. "네가 야간에 경보 분류를 할 수 있어?"

"... 못 하는 건 아니지만."

"지난번에 level 3 이벤트를 level 8로 잘못 분류해서 새벽 3시에 나 깨운 거 기억하지?"

"그건 **한 번**이잖아."

"두 번이었어."

민재가 커피를 홀짝이며 물러났다. "그래, 네가 해."

도윤이 웃었다. "민재 선배, 커피라도 맛있게 타세요."

"내 커피가 맛없다는 거야?"

"아뇨, 그냥... **일관성이 부족**하다는 거예요."

관제실에 웃음이 퍼졌다. 무거운 한 주의 끝을 웃음으로 마무리하려는 것처럼.

---

하지만 김영훈의 표정은 무거웠다.

오후에 강혜윤 실장과 통화했다. genesis의 존재를 알렸다. 강혜윤은 10초 동안 침묵했다. 그리고 말했다.

*"월요일에 만나요. 이사회 보고 방법을 논의합시다."*

이사회 보고. 서문기 상무의 머리를 넘어, 넥서스AI의 최고 의사결정 기구에 직접 보고한다는 것. 이것은 **전쟁 선포**에 가까웠다.

김영훈은 관제실을 둘러보았다. 민재가 커피를 망치고, 정규가 로그를 스크롤하고, 도윤이 가람에게 뭔가를 가르치고, 진호가 OSINT 도구를 정리하고, 지후가 포렌식 보고서를 작성하고.

*이 사람들을 지켜야 한다.*

*genesis가 세상에 나오면, 서문기는 반격할 것이다.*
*보안팀을 해체하려 할 수도 있다.*
*김영훈 개인을 공격할 수도 있다.*
*하지만 40,132건의 evidence는 그 어떤 반격보다 무겁다.*

---

### 추박하의 마지막 지시

밤 8시. 관제실에 추박하, 김영훈, 가람만 남았다.

추박하가 터미널을 열었다. 이번에는 가람에게도 보이도록.

```bash
# genesis의 PoW 체인 무결성 검증
curl -s -H "X-API-Key: opsclaw-api-key-2026" \
  -H "X-Genesis-Token: genesis-2006-parkbakha-sha256:a7f3b2..." \
  "http://localhost:8000/pow/verify?project_id=prj_000000000000"
```

```json
{
  "valid": true,
  "blocks": 40132,
  "orphans": 0,
  "tampered": [],
  "chain_start": "2006-03-15T09:23:00",
  "chain_end": "2026-04-24T09:30:00",
  "integrity": "FULL — all blocks verified, no gaps, no tampering detected"
}
```

"**40,132 블록. 전부 유효. 위변조 없음. 빈틈 없음.**" 추박하가 읽었다.

"20년간의 해시 체인이 **한 번도 깨지지 않았다**는 거죠." 가람이 말했다.

"OpsClaw를 처음 만들었을 때부터 PoW를 넣은 이유가 이거야." 추박하가 말했다. "나중에 '이건 조작된 기록이다'라는 주장에 대한 **기술적 반박**. SHA-256 해시 체인은 중간에 하나라도 수정하면 뒤의 모든 블록이 깨져. 40,132개 블록을 전부 다시 채굴하지 않는 한, 위조는 불가능해."

김영훈이 물었다. "팀장님. genesis의 **백업**은?"

"세 곳에 있어."

추박하가 화이트보드에 적었다.

```
[genesis 백업 — 3중 보관]

1. opsclaw 서버 DB (PostgreSQL)
   → 원본. PoW 체인 포함.

2. 분산 SubAgent 동기화
   → secu, web, siem 각 SubAgent에 genesis 경험의 요약본 저장
   → 원본 삭제 시 교차 검증으로 삭제 탐지 가능

3. 외부 보관
   → [비공개 위치]
   → evidence_pack (JSON + 해시 목록) 형태로 오프라인 저장
   → 회사 네트워크 외부 → 회사가 접근 불가
```

"**3번**이 핵심이에요." 가람이 말했다. "회사 네트워크 외부에 있으면, 서문기 상무가 아무리 admin 권한을 써도 삭제할 수 없어요."

"맞아." 추박하가 말했다.

김영훈이 물었다. "외부 보관은 어디에?"

추박하가 웃었다. "영훈아. 그건 나만 아는 거야. 너한테 키를 줬지만, **모든 것**을 주진 않아."

"왜요?"

"**보험의 보험**이야. 나한테 무슨 일이 생기고, 너한테도 무슨 일이 생기면, 세 번째 사본이 **자동으로** 발동하도록 설계해뒀어."

"자동으로?"

"OpsClaw의 **Schedule**이야." 추박하가 말했다. "특정 조건이 충족되면 — 예를 들어 내가 30일간 genesis에 로그인하지 않으면 — 세 번째 사본이 **지정된 수신자**에게 자동 전송돼."

```bash
# genesis의 Dead Man's Switch — 스케줄 확인
curl -s -H "X-API-Key: opsclaw-api-key-2026" \
  -H "X-Genesis-Token: genesis-2006-parkbakha-sha256:a7f3b2..." \
  "http://localhost:8000/schedules?project_id=prj_000000000000"
```

```json
{
  "schedules": [
    {
      "id": "sched-genesis-deadman",
      "cron": "0 0 * * *",
      "description": "genesis 관리자 활성 확인 — 30일 미접속 시 evidence_pack 자동 전송",
      "last_check": "2026-04-24T00:00:00",
      "days_since_login": 0,
      "trigger_threshold": 30,
      "status": "active"
    }
  ]
}
```

"**Dead Man's Switch.**" 김영훈이 중얼거렸다. "사람이 비활성화되면 시스템이 자동으로 작동하는."

"추팀장님, 이건... 언제 만드신 건가요?" 가람이 물었다.

"2015년." 추박하가 말했다. "IPO 직전에. 그때 회장이 나한테 '지금까지 한 일은 잊어라'고 했거든. 그래서 만들었어. 잊지 않기 위해."

### 마지막 대화

추박하가 코트를 입었다.

"영훈아."

"네."

"월요일에 강혜윤 실장을 만나면, genesis의 **전체 replay**를 보여줘. 20년치 전부."

"전부요?"

"전부. 숨기지 마. 강혜윤은 **법을 아는 사람**이야. 전체를 봐야 어떻게 대응할지 판단할 수 있어."

"알겠습니다."

"그리고—" 추박하가 문 앞에서 멈추었다. "영훈아."

"네."

"미안해."

김영훈이 추박하를 바라보았다.

"뭐가요?"

"내가 너한테 무능한 팀장으로 보였을 거야. 만화책이나 읽고, 야근은 안 하고, 기술적 판단은 다 네한테 떠넘기고."

"..."

"하지만 내가 게을렀던 건 — **평범해 보이고 싶어서**였어. genesis를 안고 있으면서 유능해 보이면, 누군가가 의심해. '저 사람 뭔가 숨기고 있다'고."

"그래서 **일부러** 게으르게 보인 건가요?"

추박하가 웃었다. 가람이 본 것 중 가장 편안한 웃음이었다.

"원래 좀 게을러. 그건 사실이야. 하지만 **만화책 뒤에서** 모니터를 보고 있었던 것도 사실이고."

김영훈이 작게 웃었다. "알고 있었습니다."

"알고 있었어?"

"만화책 각도가 항상 모니터 쪽으로 기울어져 있었으니까요."

추박하가 헛웃음을 쳤다. "역시 너는 관찰력이 좋아."

---

추박하가 나갔다.

김영훈이 모니터를 바라보았다. genesis의 PoW 검증 결과가 아직 화면에 남아 있었다.

```
integrity: "FULL — all blocks verified, no gaps, no tampering detected"
```

*전체 무결. 빈틈 없음. 위변조 없음.*

김영훈이 중얼거렸다.

"추 팀장. 당신이 설계한 시스템이 당신 자신을 증명하고 있어요."

### 가람의 수첩 — 4권 마지막 정리

가람은 그날 밤 집에서 수첩을 폈다. 추박하가 "적지 마"라고 했던 것은 genesis의 구체적 내용이었다. genesis가 존재한다는 사실은 적을 수 있다고 가람은 판단했다.

```
[4권 전체 정리 — "내부의 적"]

1. 이주원의 귀환
   - "스파이가 아니라 내부 고발자" 주장
   - 변호사 대동, 경찰 진술서 제출
   - 세 가지 조건 제시 (징계 철회, 고발자 인정, 외부 감사)

   - 백승현 → DataBridge LLC, 7.8억 (재무)
   - 진수현 → 윤리경영실 형식 조사 (인사)
   - 강혜윤 → Legal Hold (법무)
   - 정태식 → 패치 지연, 공유 계정 (인프라)
   - 조은별 → 기자 대응 (홍보)

   - PoW 체인 전체 무결
```
