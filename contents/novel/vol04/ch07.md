### 데이터 오염과 AI 윤리

김영훈이 화이트보드 앞에 섰다.

"기술적 관점에서 하나 더 짚겠다." 김영훈이 말했다. "불법 수집된 학습 데이터의 문제는 법적 리스크만이 아니야."

```
[데이터 오염(Data Poisoning)과 AI 윤리]

1. 편향(Bias) 증폭:
   - 경쟁사 데이터는 그 경쟁사의 편향을 그대로 포함
   - 고객 데이터의 특정 패턴이 모델에 학습됨
   → Nexus-1이 특정 고객사의 패턴에 과적합(overfitting)될 위험

2. 데이터 오염(Poisoning):
   - 출처 불명의 데이터에 악의적 패턴이 포함될 수 있음
   - darkweb_forum_crawl에는 악성 코드, 피싱 템플릿이 포함
   → 모델이 유해한 콘텐츠를 생성할 위험

3. 저작권 재생산:
   - 경쟁사의 독자적 연구 내용이 Nexus-1의 출력에 나타날 수 있음
   - "Nexus-1이 AlphaAI의 논문을 그대로 인용"하는 상황
   → 추론 시 저작권 침해 발생

4. 개인정보 유출:
   - raw 고객 데이터에서 개인 식별 정보가 학습됨
   - 프롬프트로 특정 개인의 정보를 추출할 수 있는 가능성
   → 이주원이 프롬프트 인젝션을 시도한 이유?
```
