# Chapter 1: 20년

---

월요일 새벽 5시 47분. 넥서스AI 사옥 12층 관제실.

추박하는 혼자였다. 야간 교대의 마지막 시간. 모니터의 푸른 빛만이 얼굴을 비추고 있었다.

책상 위에 만화책은 없었다. 대신, 낡은 USB 드라이브가 놓여 있었다. 2006년이라는 라벨이 붙어 있는.

추박하는 터미널을 열었다. genesis에 접속하는 습관적인 명령어.

```bash
docker exec -it genesis_core /bin/bash
cat /data/logs/genesis_activity.log | head -1
```

```
2006-03-15T09:23:17Z | INIT | genesis project created | user=parkbakha | note="모든 것을 기록한다"
```

"20년." 추박하가 중얼거렸다.

2006년 3월 15일. 넥서스AI 입사 1년차. 26살의 추박하는 보안팀 막내였다. 그때의 넥서스AI는 직원 200명짜리 중소 보안 솔루션 회사였고, 추박하는 방화벽 로그를 읽는 것이 일이었다.

---

### 2006년: 시작

그때는 OpsClaw도, genesis도, PoW 체인도 없었다.

있는 것은 **의문**이었다.

"박하야, 이 로그 좀 정리해." 당시 팀장이 말했다. "고객사 침해사고 대응 보고서 쓰는 거야."

추박하는 로그를 정리하다가 이상한 것을 발견했다. 고객사 침해사고 대응 보고서에 포함된 IP 중 하나가 — **넥서스AI의 테스트 서버 IP**와 같았다.

"팀장님, 이 IP가..."

"신경 쓰지 마. 테스트 환경이 겹치는 거야."

추박하는 신경을 껐다. 1년차 막내가 할 수 있는 건 없었다. 하지만 **기억은 했다.** 그리고 USB 드라이브에 해당 로그를 복사했다.

그것이 genesis의 씨앗이었다.

---

### AI 윤리 — 추박하의 관점

2026년 6월. 새벽의 관제실로 돌아왔다.

추박하는 화면에 문서 하나를 띄웠다. 자신이 작성한 것. 제목은 "기록자의 윤리".

```
[AI 윤리 프레임워크 — 기록자의 관점]

1. 투명성 (Transparency)
   - AI 시스템의 의사결정 과정은 추적 가능해야 한다
   - OpsClaw의 PoW 체인: 모든 작업이 위변조 불가능하게 기록됨
   - genesis: 투명성을 회사가 아닌 "기록자"가 보장하는 경우

2. 책임성 (Accountability)
   - AI의 행동에 대한 책임은 누구에게 있는가?
   - OpsClaw가 실행한 명령의 책임: 지시자(서문기) vs 실행자(추박하) vs 시스템(OpsClaw)
   - genesis의 40,132건: 책임의 연쇄를 추적할 수 있는 유일한 기록

3. 공정성 (Fairness)
   - AI가 편향된 데이터로 학습하면 편향된 결과를 낸다
   - Nexus-1의 학습 데이터: 불법 수집 데이터로 학습된 모델은 "공정"한가?
   - 기술적으로 뛰어나더라도, 윤리적으로 결함이 있으면 공정하지 않다

4. 프라이버시 (Privacy)
   - 12,567건의 customer_data_ops — 고객 데이터를 학습에 사용
   - 개인정보 비식별화 없이 ETL 파이프라인에 투입된 경우
   - GDPR, 개인정보보호법 위반 가능성

5. 안전성 (Safety)
   - AI 시스템이 의도하지 않은 결과를 초래하지 않아야 한다
   - genesis의 Dead Man's Switch: 안전장치인가, 위협인가?
   - 30일 미접근 시 자동 발동 → 의도된 안전장치, 하지만 해석은 갈린다
```

추박하는 이 문서를 바라보았다. 자신이 20년간 해온 것을 5가지 항목으로 정리할 수 있다는 것이 — 우스웠다.

### 2010년: 첫 번째 지시

"박하야." 당시 이사였던 서문기가 불렀다. "경쟁사 A사의 차세대 방화벽 스펙이 필요해."

"합법적인 경로로요?"

서문기가 웃었다. "합법적인 경로로 구할 수 있으면 내가 왜 너한테 부탁하겠어."

추박하는 A사의 개발 서버에 침투했다. SQL Injection. 2010년의 A사는 보안이 허술했다. 문서를 가져오는 데 3시간이 걸렸다.

genesis에 기록했다.

```bash
# 2010년의 기록 (genesis evidence)
curl -X POST http://localhost:8000/projects/prj_000000000000/dispatch \
  -H "Content-Type: application/json" \
  -H "X-API-Key: $OPSCLAW_API_KEY" \
  -d '{
    "command": "record_evidence",
    "params": {
      "category": "competitive_intelligence",
      "action": "unauthorized_access",
      "target": "CompanyA_dev_server",
      "method": "SQL_injection",
      "ordered_by": "seo_mungi",
      "evidence_hash": "sha256:a3f7b2..."
    }
  }'
```

물론 2010년에는 OpsClaw도, Manager API도 없었다. 추박하는 SQLite 데이터베이스에 직접 INSERT했다. OpsClaw가 완성된 후에야 genesis의 모든 레코드를 PoW 체인으로 마이그레이션했다.

"기록하지 않으면 아무도 모른다." 추박하가 26살에 USB에 로그를 복사하면서 했던 말. 그것이 20년간의 원칙이 되었다.

### 2015년: 분기점

넥서스AI가 코스닥에 상장했다. 직원이 1,500명으로 늘었다. 추박하는 보안팀장이 되었다.

서문기는 상무가 되었다.

"추 팀장." 서문기가 말했다. "이제 우리는 상장사야. **보안 수준을 올려야 해.** ISO 27001 인증 준비를 해."

추박하는 ISO 27001 인증을 준비하면서, 동시에 서문기의 지시로 경쟁사 B사의 고객 리스트를 빼오는 작전을 수행했다.

인증 컨설턴트가 말했다. "넥서스AI의 보안 체계는 인상적이네요. 특히 접근 제어 정책이 잘 되어 있습니다."

추박하는 고개를 숙였다. 자신이 설계한 접근 제어 정책은 — 자신이 **우회**하는 데 가장 효율적인 구조이기도 했다.

genesis에 기록했다. 인증 컨설턴트의 칭찬도, 그 칭찬이 얼마나 아이러니한지도.

---

### 2020년: OpsClaw

김영훈이 입사한 해.

추박하는 OpsClaw를 설계했다. 공식적으로는 "보안 자동화 도구". 실제로는 — **genesis를 보호하기 위한 껍데기.**

OpsClaw의 PoW 체인은 genesis의 evidence를 위변조 불가능하게 만드는 메커니즘이었다. 추박하는 자신이 기록한 것이 언젠가는 필요해질 것이라고 — 알고 있었다.

```
[AI 윤리와 인간 윤리의 교차]

AI 윤리는 말한다:
  "AI 시스템은 투명하고, 책임감 있고, 공정해야 한다"

인간 윤리는 묻는다:
  "불의를 목격한 사람은 무엇을 해야 하는가?"
  "지시에 따른 것은 면죄부가 되는가?"
  "기록하는 것은 저항인가, 공모인가?"

추박하의 20년:
  지시를 따랐다 → 공범
  기록을 남겼다 → 증인
  두 가지가 동시에 성립한다

Genesis의 딜레마:
  증거를 제출하면 → 회사의 비리가 밝혀진다
  증거를 제출하면 → 자신의 범죄도 밝혀진다
  증거를 제출하지 않으면 → 20년간의 기록이 무의미해진다
```

### 현재: 새벽 6시 30분

교대 시간이 다가왔다. 추박하는 genesis의 터미널을 닫았다.

복도에서 발소리가 들렸다. 김영훈이었다. 이 시간에 출근하는 것은 이례적이었다.

"... 추팀장님, 왜 여기 계세요?" 김영훈이 물었다.

"야간 교대 마무리." 추박하가 말했다.

"추팀장님이 야간 교대를 서신 적은 없잖아요."

추박하가 웃었다. 피곤한 웃음이었다.

"오늘은 좀 **생각할 게** 있어서."

김영훈이 추박하의 책상을 보았다. USB 드라이브. 2006이라는 라벨.

"그건 뭡니까?"

"**genesis의 씨앗.**" 추박하가 말했다. "20년 전에 처음 기록을 시작한 USB야. 언젠가 네가 이걸 이해할 날이 올 거라고 생각했어."

김영훈은 대답하지 않았다. 하지만 추박하의 눈 아래 그림자를 보았다. 20년의 무게가 거기에 있었다.

---

오전 8시. 관제실에 팀원들이 하나둘 들어왔다.

"어? 추팀장님이 **먼저** 출근하셨네요?" 민재가 놀랐다.

"세상이 끝나려나 봐." 도윤이 말했다.

"어제 야근이야." 추박하가 말했다.

"야근이요? 추팀장님이?" 민재가 더 놀랐다. "8년간 야근하신 적 없으시잖아요."

"올해는 좀 **특별한 해**니까." 추박하가 말했다. 만화책을 꺼내는 대신 커피를 마셨다.

가람이 조용히 수첩을 펼쳤다. 추팀장의 변화가 가속되고 있었다. 만화책이 사라지고, 새벽 근무가 나타나고, 넥타이를 매는 날이 늘었다.

"뭔가 큰 게 오고 있어요." 가람이 속삭였다. 정규에게.

정규가 고개를 끄덕였다. "안다."
