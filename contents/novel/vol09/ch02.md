### CTI 심화 — 위협 지형도

송진호가 OpenCTI 대시보드를 열었다.

```
[CTI 심화 — 위협 지형도 구성]

1. 전술적 CTI (Tactical)
   - IOC: IP, 도메인, 해시
   - 방어 장비에 즉시 적용 가능
   - 유효 기간: 수시간 ~ 수일

2. 운용적 CTI (Operational)
   - TTP: 공격자의 전술, 기법, 절차
   - ATT&CK 매핑으로 공격 패턴 파악
   - 유효 기간: 수주 ~ 수개월

3. 전략적 CTI (Strategic)
   - 공격자의 동기, 역량, 목표
   - 경영진 보고용 — 비즈니스 리스크와 연계
   - 유효 기간: 수개월 ~ 수년

4. 피라미드 of Pain (David Bianco)
   Hash → IP → Domain → Network/Host Artifacts → Tools → TTPs
   ↑ 쉽게 바꿈                                    어렵게 바꿈 ↑
   → 상위 단계를 탐지할수록 공격자에게 더 큰 고통을 줌
```

"우리가 지금 하는 것은 **전략적 CTI**야." 김영훈이 말했다. "ModelThief가 **누구인지, 왜 넥서스AI를 노리는지, 다음에 무엇을 할지** 파악하는 거지."

### AlphaAI 연결 분석

김영훈이 genesis의 데이터를 OpenCTI에 투입한 결과를 보여주었다.

```bash
# genesis competitive_intelligence와 ModelThief IOC 교차 분석
docker exec genesis_core python3 -c "
import sqlite3, json
conn = sqlite3.connect('/data/genesis.db')

# genesis에 기록된 경쟁사 관련 IP 추출
genesis_ips = set()
for row in conn.execute('''
    SELECT json_extract(payload, '$.target_ip')
    FROM evidence
    WHERE category = 'competitive_intelligence'
    AND json_extract(payload, '$.target_ip') IS NOT NULL
'''):
    genesis_ips.add(row[0])

# ModelThief C2 서버 IP (OpenCTI에서 추출)
modelthief_ips = {
    '198.51.100.23', '203.0.113.45', '192.0.2.78',
    '198.51.100.91', '203.0.113.112'
}

overlap = genesis_ips & modelthief_ips
print(f'genesis IP: {len(genesis_ips)}개')
print(f'ModelThief IP: {len(modelthief_ips)}개')
print(f'겹치는 IP: {len(overlap)}개')
for ip in overlap:
    print(f'  → {ip}')
"
```

```
genesis IP: 347개
ModelThief IP: 5개
겹치는 IP: 1개
  → 203.0.113.45
```

관제실이 조용해졌다.

"**하나가 겹쳐.**" 진호가 말했다. "203.0.113.45."

"이 IP가 genesis에 어떻게 기록되어 있는지 확인하자." 김영훈이 말했다.

```bash
docker exec genesis_core python3 -c "
import sqlite3, json
conn = sqlite3.connect('/data/genesis.db')
for row in conn.execute('''
    SELECT timestamp, json_extract(payload, '$.action'),
           json_extract(payload, '$.ordered_by'),
           json_extract(payload, '$.note')
    FROM evidence
    WHERE category = 'competitive_intelligence'
    AND json_extract(payload, '$.target_ip') = '203.0.113.45'
    ORDER BY timestamp
'''):
    print(f'{row[0]} | {row[1]} | ordered_by={row[2]} | {row[3]}')
"
```

```

2019-08-01T16:45:00Z | data_exfil | ordered_by=seo_mungi | AlphaAI 차세대 모델 스펙 탈취
```

```
[AlphaAI-ModelThief 연결 가설]

가설 A: 우연의 일치
  - 같은 호스팅 업체의 IP가 재활용된 것
```

### MISP/OpenCTI 실습

진호가 팀원들에게 CTI 심화 실습을 진행했다.

```bash
# OpenCTI에서 ModelThief IOC를 STIX 번들로 내보내기
curl -X POST http://opencti:8080/graphql \
  -H "Authorization: Bearer $OPENCTI_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "query { stixCoreObjects(filters: { key: \"objectLabel\", values: [\"ModelThief\"] }) { edges { node { standard_id entity_type ... on Indicator { pattern } } } } }"
  }' | python3 -m json.tool
```

```bash
# MISP와 연동 — IOC 자동 공유
# OpenCTI → MISP connector로 위협 정보를 커뮤니티에 공유
# (넥서스AI 관련 민감 정보는 TLP:RED로 제한)

# TLP(Traffic Light Protocol):
#   TLP:RED    — 수신자만 열람
#   TLP:AMBER  — 수신자 조직 내 공유
#   TLP:GREEN  — 커뮤니티 공유 가능
#   TLP:WHITE  — 무제한 공유
```

"ModelThief의 IOC는 **TLP:GREEN**으로 공유해서 다른 AI 기업도 방어할 수 있게 하고," 진호가 말했다, "넥서스AI의 내부 정보는 **TLP:RED**로 제한해야 해."
