### 한승우의 최종 보고서

한승우가 빔 프로젝터를 켰다. **SecureEdge의 최종 보고서.**

```
[Operation Shield — 최종 보고서 (SecureEdge)]

기간: 2026.05.12 ~ 05.23, 14일간
Red Team: 89회 공격 시도 (초기 47 + 후반 42)

취약점 총 발견:
  Critical: 3건  (Nexus-1 API 무인가, SubAgent 무인가, API 백도어)
  High:     6건
  Medium:   5건
  Low:      4건
  → 총 18건

Blue Team 최종 평가:
  탐지율:     94% (89건 중 84건)
  대응 시간:  평균 11분 (Critical 평균 6분)
  차단 성공:  78/84 (93%)
  자동화 수준: 높음 (OpsClaw + Blue Agent)
  → 종합 등급: A- (우수)
```

"**A-?**" 서문기가 의외라는 표정을 지었다.

"Blue Team의 탐지/대응 능력은 업계 상위 10% 수준입니다." 한승우가 말했다. "특히 **OpsClaw 기반 자동화**와 **LLM 경보 분석**은 제가 본 SOC 중에서 가장 진보적입니다."

김영훈의 얼굴에 미소가 아닌 **안도**가 스쳤다.

"하지만 A+가 아닌 이유는—" 한승우가 말을 이었다.

```
[A+가 아닌 이유]

1. Nexus-1 API 인증 부재 (Critical, 미조치)
   → 보안팀이 보고했으나 경영진이 미조치
   → 책임: AI 연구소 / 서문기 상무

2. SubAgent 인증 부재 (Critical, 14일차에야 nftables로 보완)
   → 원래 인증이 설계되어야 했음
   → 책임: 보안팀(설계), 인프라팀(구현)

3. JuiceShop 네트워크 격리 미완 (High)
   → 테스트 앱이 프로덕션 네트워크에 존재
   → 책임: 인프라팀
```

"3개 중 **1번은 보안팀 책임이 아닙니다.**" 한승우가 명확하게 말했다. "보안팀은 보고했으나, **경영진이 조치하지 않은** 것입니다."

서문기의 얼굴이 붉어졌다. 대회의실의 모든 시선이 서문기에게 향했다.

"한 대표." 서문기가 말했다. "그 부분은 **내부 사안**이니까 보고서에서—"

"**독립 감사 보고서**에서 사실을 빼는 것은 불가능합니다." 한승우가 차분하게 말했다. "발견된 모든 사실을 있는 그대로 보고하는 것이 SecureEdge의 원칙입니다."

강혜윤이 개입했다. "한 대표님의 보고서는 **SOC 2 감사 자료**로 사용됩니다. 사실이 빠지면 감사의 신뢰성이 훼손됩니다. 있는 그대로 보고해주세요."

서문기가 입을 다물었다. 반박할 수 없었다.

### 특별 섹션: 실제 APT 발견

한승우가 보고서의 특별 섹션을 열었다.

"모의해킹 기간 중 **실제 APT 공격**이 발생했습니다."

```
[실제 APT 발견 — Day 5]

발견: Blue Team (민재, 지후) — 새벽 3:17
공격자: ModelThief (ESET 추정)
C2: 45.33.32.156 (Linode), Cobalt Strike Beacon
진입: JuiceShop 파일 업로드 취약점
대응: 3분 만에 탐지 + 프로세스 종료 + IP 차단
결과: 데이터 유출 없음

SecureEdge 평가:
  "실제 APT를 모의해킹 노이즈 속에서 3분 만에 탐지한 것은
   매우 뛰어난 대응입니다. Blue Team의 Red Team IP 식별 전략과
   OpsClaw의 자동 경보 분석이 결합된 결과입니다."
```

"이건 보안팀이 자랑할 만한 성과입니다." 한승우가 말했다. 처음으로 미소를 보였다.

"**3분 대응.**" 정태식이 중얼거렸다. "인프라팀이 네트워크 장애에 대응하는 것보다 빨라."

민재가 작게 주먹을 쥐었다. 새벽 3시에 깨어 있었던 보상이었다.

### 추박하의 한 마디

보고가 끝나고, 질의응답 시간.

추박하가 손을 들었다. 관제실 전원이 놀랐다. 추박하가 **공식 석상에서 발언**하는 것은 거의 없었다.

"한 대표님. 좋은 보고서 감사합니다."

"감사합니다." 한승우가 고개를 숙였다.

"한 가지 질문. 이 보고서의 **수신자**는 누구입니까?"

"서문기 상무님과 강혜윤 실장님입니다."

"이사회에는?"

한승우가 잠시 멈추었다. "의뢰자의 요청에 따라 결정됩니다."

추박하가 서문기를 바라보았다. 그리고 강혜윤을 바라보았다.

"강 실장. 이 보고서를 **이사회에 제출**하시는 게 좋겠습니다."

강혜윤이 고개를 끄덕였다. "동의합니다. SOC 2 감사 대비 자료로 이사회에 보고하겠습니다."

서문기의 손이 테이블 아래에서 떨리고 있었다.

### 회의 후: 복도에서

회의가 끝나고 복도에서 한승우가 추박하에게 다가왔다.

"추 팀장님."

"한 대표."

"오래전에 한 번 뵙겠다고 했는데, 결국 이렇게 만나네요."

추박하가 웃었다. "인연이 그런 거지."

"보고서에 한 가지 더 넣고 싶은 게 있었는데, 뺐습니다."

"뭔데?"

"**genesis 프로젝트.** OpsClaw DB에 prj_000000000000이라는 프로젝트가 있는 건 알고 있습니다. 접근 권한은 없었지만."

추박하의 표정이 변하지 않았다. "알고 있었어?"

"보안 감사를 하면서 DB 스키마를 봤습니다. prj_000000000000은 2006년에 생성된 프로젝트고, evidence가 40,000건 이상. 보안 감사 보고서에 넣기엔 **범위를 벗어나서** 뺐지만."

"고마워." 추박하가 말했다. "아직 때가 아니야."

"알고 있습니다." 한승우가 고개를 숙이며 돌아섰다. "하지만 **때가 오면**, SecureEdge가 **독립 검증**을 해드릴 수 있습니다."

한승우가 사라진 후, 추박하가 중얼거렸다.

"좋은 사람이네."

가람이 옆에서 듣고 있었다. 적지 않았다. 하지만 기억했다.

*한승우는 genesis를 알고 있다. 하지만 보고서에 넣지 않았다.*
*"아직 때가 아니다"라는 추팀장의 말을 존중한 것이다.*
*한승우는 서문기의 사람이 아니었다. 서문기의 의뢰를 받았지만, 독립성을 지켰다.*
*그리고 "때가 오면 독립 검증"을 제안했다.*

*추팀장의 체스판에 새로운 **기사(Knight)**가 추가되었다.*

### 저녁: Purple Team의 시작

Operation Shield가 끝났다. 관제실에서 간단한 뒤풀이.

추박하가 치킨을 시켰다. "오늘은 내가 쏜다."

"또요?" 도윤이 놀랐다. "CTF 때도 쏘셨잖아요."

"2주간 고생했으니까." 추박하가 말했다. "그리고 민재."

"네."

"새벽 3시에 깨어 있어줘서 고마워."

민재의 눈이 살짝 붉어졌다. "... 감사합니다."

치킨을 먹으면서 김영훈이 말했다.

"모의해킹은 끝났지만, **보안은 끝나지 않아.** 다음 단계는 **Purple Team** — 공격과 방어를 상시적으로 반복하는 체계야."

"OpsClaw의 자율 Red/Blue Agent?" 정규가 물었다.

"맞아. 7장에서 설계한 것을 **실제로 구현**하자. Red Agent가 자동으로 공격하고, Blue Agent가 자동으로 방어하고, 양쪽 다 Experience를 쌓으면서 진화하는."

"**자기강화형 보안 시스템.**" 가람이 말했다.

"7권에서 본격적으로." 김영훈이 웃었다.
