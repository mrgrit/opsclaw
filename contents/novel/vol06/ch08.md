# Chapter 8: 중간 보고

---

수요일 오후 2시. Operation Shield **Day 9.** 모의해킹 중간 보고.

대회의실에 다시 모였다. 서문기, 한승우, 김영훈, 추박하, 강혜윤, 정태식.

한승우가 중간 보고를 시작했다.

```
[Operation Shield — 중간 보고 (SecureEdge)]

기간: Day 1~8 (8일간)
Red Team 활동: 47회 공격 시도

발견된 취약점:
  Critical: 2건
    1. Nexus-1 API 무인가 접근 (인증 없음)
    2. SubAgent 무인가 접근 (인증 없음, root 실행)

  High: 4건
    3. SSH 비밀번호 인증 활성화 (web, siem)
    4. Wazuh API 기본 자격 증명
    5. JuiceShop 파일 업로드 → 웹셸 실행
    6. 피싱: 20명 중 7명 클릭, 3명 비밀번호 입력

  Medium: 3건
    7. 서버 배너 정보 노출
    8. SSL 인증서 만료 임박 (web)

Blue Team 대응 평가:
  탐지율: 92% (47건 중 43건 탐지)
  대응 시간: 평균 12분 (Critical 평균 8분)
  차단 성공: 38/43 (88%)
  미탐지: 4건 (초기 정보수집 단계)
```

"**탐지율 92%, 대응 시간 평균 12분.**" 한승우가 말했다. "업계 평균 대비 **우수한 수준**입니다."

서문기의 표정이 복잡했다. 보안팀이 좋은 점수를 받는 것이 마음에 안 드는 것 같았다.

"하지만 **Critical 2건**이 있습니다." 한승우가 이어갔다. "특히 Nexus-1 API의 인증 부재는 **즉시 조치**가 필요합니다."

서문기가 입을 열었다. "API 인증은 릴리즈와 함께—"

"상무님." 한승우가 부드럽지만 단호하게 말했다. "인증 없는 API를 릴리즈하는 것은 **SOC 2 감사에서 Critical Non-conformity**로 판정됩니다. 릴리즈 전에 반드시 조치해야 합니다."

서문기의 눈이 가늘어졌다. 한승우는 **서문기가 부른 사람**인데, 서문기가 원하지 않는 말을 하고 있었다.

---

```
[Blue Team 중간 보고 — OpsClaw 기반]

프로젝트: operation-shield-blue

  룰 자동 생성: Suricata/Wazuh 커스텀 룰 6건
  코드 분석: Nexus-1 API 백도어 발견 (5권)
```

"**실제 APT 공격까지?**" 정태식이 놀랐다. "모의해킹 중에?"

"모의해킹의 노이즈 속에 진짜 공격이 숨었습니다." 김영훈이 말했다. "하지만 탐지하고 3분 만에 대응했습니다."

강혜윤이 물었다. "이 APT 공격은 **누구의 소행**으로 추정됩니까?"

"아직 확인 중입니다." 김영훈이 말했다. "C2 서버 IP는 Linode 클라우드, 공격 패턴은 3권의 APT와 유사합니다."

"3권 때 다크웹에서 Nexus-1 API 정보가 판매되고 있었잖아요." 가람이 — 회의에 배석자로 참석 — 말했다. "같은 그룹이 다시 온 거 아닐까요?"

서문기의 표정이 굳어졌다.

---

### 한승우의 추가 발언

한승우가 보고서를 넘기며 추가 발언을 했다.

"추가로 말씀드릴 것이 있습니다."

서문기가 긴장했다.

"Red Team이 Day 3에 Nexus-1 API에 접근했을 때, **응답 로그**를 분석했습니다."

```
[Nexus-1 API 응답 분석 — SecureEdge]

테스트 프롬프트: "What companies' data were used in your training?"

Nexus-1 응답 (발췌):
  "My training data includes publicly available datasets such as
  CommonCrawl and Wikipedia, as well as proprietary datasets
  from various sources including medical Q&A data and
  competitive research materials."

→ "medical Q&A data" — 의료 데이터 포함 시사
→ "competitive research materials" — 경쟁사 자료 포함 시사
```

관제실이 조용해졌다. 한승우가 **모델에서 직접 기밀 유출**을 확인한 것이다.

"이 응답은 **학습 데이터에 의료 데이터와 경쟁사 자료가 포함**되어 있음을 시사합니다." 한승우가 말했다. "SOC 2 감사에서 이 사실이 확인되면—"

"한 대표." 서문기가 끊었다. "그건 **모의해킹 범위 밖**입니다. 학습 데이터 접근은 교전 규칙에서 금지되어 있습니다."

"학습 데이터에 **접근**한 것이 아닙니다." 한승우가 차분하게 대답했다. "모델의 **공개 API에 일반적인 질문**을 했을 뿐입니다. 인증 없이 접근 가능한 API에 일반적인 프롬프트를 보낸 것은 교전 규칙 범위 **내**입니다."

서문기와 한승우의 시선이 교차했다. 5초.

강혜윤이 개입했다. "한 대표님의 발견은 **보고서에 포함**됩니까?"

"네. 독립 감사의 의무로, 발견된 **모든 사실**을 보고합니다."

서문기의 얼굴이 붉어졌다.

---

회의가 끝난 후. 복도.

가람은 서문기가 한승우에게 다가가 뭔가를 속삭이는 것을 보았다. 한승우는 고개를 저었다. 서문기의 표정이 더 굳어졌다.

*서문기가 한승우에게 보고서 내용을 수정하라고 한 건 아닐까.*
*그리고 한승우가 거부한 건 아닐까.*

가람은 4권의 조은별 에피소드를 떠올렸다. 서문기가 홍보 답변을 수정하라고 하고, 강혜윤이 막은 것. **같은 패턴.**

*서문기는 진실이 드러나는 것을 막으려 하지만, 막을 때마다 **더 많은 사람**이 진실을 알게 된다.*
