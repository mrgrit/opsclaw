"네트워크 측면은?" 김영훈이 진호를 바라보았다.

진호가 Suricata 로그에서 C2 외의 비정상 트래픽을 검색했다.

```bash
# ICMP 이상 트래픽 (터널링 의심)
sshpass -p1 ssh -o StrictHostKeyChecking=no secu@10.20.30.1 \
  "echo 1 | sudo -S cat /var/log/suricata/eve.json 2>/dev/null | \
   python3 -c \"
import sys, json
for l in sys.stdin:
    try:
        e = json.loads(l)
        if e.get('event_type') == 'alert' and 'ICMP' in str(e.get('alert',{}).get('signature','')):
            print(f'{e[\"timestamp\"]} | {e[\"alert\"][\"signature\"]}')
    except: pass
\" 2>/dev/null | head -5"
```

```
2026-04-02T01:15:22 | ET POLICY ICMP Oversized Payload (1400+ bytes)
2026-04-02T01:15:23 | ET POLICY ICMP Oversized Payload (1400+ bytes)
2026-04-02T01:15:24 | ET POLICY ICMP Oversized Payload (1400+ bytes)
```

"**대용량 ICMP 패킷.** 1,400바이트 이상." 진호가 말했다. "정상 ping은 64바이트야. 1,400바이트는 **ICMP 터널링**이야."

"데이터를 ICMP 패킷 안에 숨겨서 빼가는 거야." 김영훈이 설명했다. "방화벽이 ICMP(ping)를 허용하고 있으면, 이 경로로 데이터 유출이 가능해."

"nftables에서 ping은 허용되어 있었죠." 정규가 확인했다.

"맞아. `icmp type echo-request accept`. 하지만 **페이로드 크기 검사**는 안 하고 있었어."

"**완전한 킬체인(Kill Chain)**이야." 진호가 말했다.

```
정찰 → 침투 → C2 구축 → 내부 정찰 → 권한 유지 → 데이터 유출 → 추가 공격
```

"MITRE ATT&CK으로 매핑하면 최소 **10개 이상의 기법**이 사용됐어."
