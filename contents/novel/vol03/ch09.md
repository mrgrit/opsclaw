# Chapter 9: 연결 고리

---

월요일. 시큐리온 팀의 최종 분석이 나왔다.

"이주원, 퇴직자(김성민), AlphaAI. **세 점을 잇는 선**을 찾았습니다."

위협 헌팅 전문가가 대형 모니터에 관계도를 띄웠다.

```
┌──────────────────────────────────────────────────┐
│                                                  │
│  넥서스AI                    AlphaAI             │
│  ┌──────┐   이직   ┌──────┐   재직   ┌──────┐  │
│  │김성민│ ──────→  │김성민│ ──────→  │김지혜│  │
│  │(퇴직)│          │(전직) │         │(연구원)│  │
│  └──┬───┘          └──────┘          └──┬───┘  │
│     │                                    │      │
│     │ 동기                               │      │
│     │                                    │      │
│  ┌──┴───┐   Google Drive / Mega   ┌──────┐     │
│  │이주원│ ◄──────────────────────│  ???  │     │
│  │(실종)│                         │ (jw?) │     │
│  └──────┘                         └──────┘     │
│                                                  │
│  jw.research.2025@gmail.com = 이주원의 외부 계정  │
│  퇴직자 PC에서 AlphaAI 보고서(김지혜 작성) 발견   │
│  이주원 PC에서 Nexus-1 데이터 → Mega 업로드 확인  │
└──────────────────────────────────────────────────┘
```

"김성민은 넥서스AI에서 3년 근무 후, 2025년 8월에 퇴직했어요. 퇴직 후 AlphaAI에 이직한 것으로 **추정**됩니다."

"추정?" 김영훈이 물었다.

"LinkedIn 프로필에 AlphaAI가 나오지만, 공식 확인은 안 됐어요. 하지만 퇴직자 노트북에서 발견된 AlphaAI 보고서(김지혜 작성)가 김성민의 Google Drive 링크를 통해 공유된 것으로 보입니다."

---

"그러면 이야기가 이렇게 되는 건가요?" 진호가 정리했다.

```
1. 김성민이 넥서스AI에서 AlphaAI로 이직
2. 김성민이 AlphaAI에서 동료(김지혜)의 내부 보고서를 확보
3. 김성민이 이 보고서를 넥서스AI의 이주원에게 전달 (왜?)
4. 이주원이 넥서스AI 내부에서 Nexus-1 데이터를 수집
5. 이주원이 수집한 데이터를 Mega(jw.research.2025)에 업로드
6. 이 데이터의 최종 수신자는 누구? AlphaAI? 다른 곳?
```

"3번이 핵심이야." 김영훈이 말했다. "김성민이 왜 AlphaAI 보고서를 이주원에게 줬을까?"

시큐리온 전문가가 대답했다. "두 가지 가능성. **A: 이주원이 AlphaAI에 이직을 준비**하고 있었고, 김성민이 '이런 게 있으니 참고해'라고 공유. **B: 조직적인 산업 스파이** — 김성민이 AlphaAI의 요청으로 넥서스AI 내부에 남은 이주원을 활용."

"B가 맞으면 **AlphaAI가 배후**가 되는 건데..." 민재가 말했다.

"증거가 부족해." 추박하가 말했다. "이주원을 찾아서 **직접 확인**하기 전까지는 추정이야."

---

"이주원의 행방은?" 강혜윤이 물었다. 그녀도 오늘 참석해 있었다.

"경찰에 실종 신고가 접수되었고, 출입국 기록을 확인 중이에요." 진수현이 대답했다. "현재까지 **출국 기록은 없어요**."

"그러면 국내에 있다는 뜻이네요."

"아마도. 하지만 핸드폰은 꺼져 있고, 카드 사용 기록도 없어요."

"**의도적으로 추적을 피하고 있다**는 뜻이야." 추박하가 말했다.

---

강혜윤이 법적 상황을 정리했다.

```
[법적 현황]
1. 이주원 — 부정경쟁방지법 위반 (영업비밀 유출) → 경찰 수사 중
2. 김성민 — AlphaAI 이직 확인 시, 전직금지 조항 위반 가능
3. AlphaAI — 넥서스AI 영업비밀 사용이 확인되면 부정경쟁방지법 위반
4. 넥서스AI — 학습 데이터 문제가 확인되면 별도 법적 리스크

⚠️ 4번이 가장 민감. 현재 내부 확인 중.
```

"4번." 김영훈이 말했다. "학습 데이터 문제."

강혜윤이 추박하를 바라보았다. 두 사람의 시선이 만났다.

"추 팀장님. 학습 데이터에 대해 **아시는 게** 있나요?"

추박하가 잠시 침묵했다. 관제실의 모든 시선이 그에게 집중되었다.

"... 있어."

김영훈이 앞으로 몸을 기울였다.

"하지만 **지금은 아직** 말할 때가 아니야." 추박하가 말했다. "이주원을 먼저 찾아. 그가 뭘 알고 뭘 갖고 나갔는지 확인한 다음에."

"팀장님—"

"영훈아." 추박하의 눈이 진지했다. "**순서를 지켜.** 급하면 실수해."

---

가람은 수첩에 적었다.

> `연결 고리: 김성민(퇴직→AlphaAI) — 이주원(현직→실종) — AlphaAI`
> `추팀장: "학습 데이터에 대해 아는 게 있다" — 하지만 "지금은 아직 아니다"`
> `이것은 처음으로 추팀장이 비밀의 존재를 **인정**한 순간이다.`
> `"순서를 지켜. 급하면 실수해." — 누구의 실수? 우리의? 아니면 추팀장의?`
