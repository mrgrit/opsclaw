## 아이디어: 자율 공방의 수렴 한계와 계층적 보안 운용 모델

### 핵심 연구 질문
- LLM 에이전트 기반 Red/Blue 무한 루프가 보안을 완전하게 만들 수 있는가?
- 수렴 한계(로컬 최적값)를 극복하려면 어떤 계층적 구조가 필요한가?

### 주장
무한 루프는 모델 능력 천장에서 수렴하며 "거짓 안심감"을 줄 수 있다.
이를 극복하는 **계층적 보안 순환 모델** 제안:

```
Layer 1: 경량 모델 (8B/12B) — 상시 순환 (자동, 저비용)
Layer 2: 중형 모델 (32B/70B) — 주간 순환 (더 복잡한 공격)
Layer 3: 상용 대형 모델 (Claude/GPT-4) — 월간/이벤트 트리거 (새 CVE, 인프라 변경)
Layer 4: 인간 전문가 — 분기별 (사회공학, 물리, 공급망)
```

각 계층이 발견한 것을 Playbook화하여 하위 계층에 전파 → 자동화 범위 점진 확대

### 필요한 실험
1. 경량 모델 N회 반복 → 수렴 곡선 측정 (몇 라운드에서 새 발견이 0이 되는가)
2. 동일 시나리오를 8B → 32B → 70B → Claude로 실행 → 계층별 발견 차이
3. 하위 계층에 상위 경험 전파 후 재실행 → 개선 효과 측정
4. 새 CVE 주입 시 각 계층의 반응 속도

### 대상 학회
보안/AI 학회: USENIX Security, ACSAC, NDSS, 또는 AI Safety 워크숍
